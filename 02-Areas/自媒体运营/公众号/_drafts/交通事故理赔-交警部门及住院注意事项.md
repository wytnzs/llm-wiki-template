# 发生交通事故后，交警部门和住院你要知道的那些事

出了交通事故，人进了医院，事情才刚开始。

很多受害人以为等伤好了找对方要钱就行了。实际上，从交警部门到住院治疗，每一步都藏着"坑"——踩一个，赔偿可能少几万。

今天把这两件事分开讲清楚。

---

## 一、在交警部门办事的五个关键点

### 1. 配合交警办案，不是被动配合

很多人见到交警就紧张，问什么答什么，问不到就不说。

这是大忌。

交通事故责任认定，直接决定你的赔偿数额。同一种情形，可以认定一方主要责任，也可以认定双方同等责任——**差一个级别，赔偿可能差出一大半**。

所以你要做的是：

- 积极主动陈述对自己有利的事实
- 收集对方不利的证据
- 指出对方违规的细节

不知道怎么找对自己有利的事实？

可以找专业人士协助。**交警队的责任划分一旦签字确认，后期想推翻很难。**

---

### 2. 这几份资料必须拿到：三证一单

在交警队，有几份资料是必须复印到手的：

| 资料名称 | 为什么要拿 |
|---------|------------|
| 肇事车辆行驶证 | 法院立案必备 |
| 肇事司机驾驶证 | 法院立案必备 |
| 肇事司机身份证 | 确认身份信息 |
| 肇事车辆保险单 | 确认保险情况 |

还有一个**最最重要的**——

**交通事故认定书原件，必须自己保留。**

有人告诉你"要把原件交给保险公司才能理赔"，这是错的。认定书原件一旦交出去，举证难度会大幅增加。

---

### 3. 车辆扣留：最长多久？什么时候该行动？

**扣车的合法情形**

根据规定，以下8种情况可以扣留车辆：

1. 未悬挂号牌、未放置检验标志 / 保险标志、未携带行驶证 / 驾驶证
2. 伪造 / 变造号牌 / 行驶证 / 检验标志 / 保险标志
3. 未投保交强险
4. 客运 / 货运车辆超载
5. 机动车有盗抢嫌疑
6. 机动车有拼装 / 达到报废标准嫌疑
7. 未申领剧毒化学品运输通行证
8. 非机动车驾驶人拒绝接受罚款处罚

**扣车时间：正常46天**

| 步骤 | 时间 |
|------|------|
| 交警3日内决定是否立案 | 第1-3天 |
| 立案后3日内委托鉴定 | 第4-6天 |
| 鉴定机构30日内完成鉴定 | 第7-36天 |
| 交警5日内送达鉴定报告 | 第37-41天 |
| 通知当事人取车 | 第42-46天 |

**超过30天需报上级批准，最长不超过60天。**

**你需要做什么？**

在交警通知取车之前，你得**向法院申请查封扣押这辆车**。

很多人不知道这步，等去法院申请的时候，交警那边已经放车了——对方把车提走，财产保全落空，赔偿执行难上难。

法院收到申请后，一般需要7-15天才会采取行动。**这个时间差，就是你要抓住的窗口期。**

---

### 4. 要医药费：事故认定书出来前最有效

受害人住院，最急的是钱。

如果你的诉前保全申请还没通过，这时候找交警打电话催医药费，**在责任认定书出来之前，交警的电话还是很管用的。**

认定书出来之后呢？

交警一般就无能为力了。

所以这段时间，要做两件事：

- 了解对方车主的财产状况
- 防止对方转移财产

怎么了解？

通过办案交警，索要对方的**三证一单**——行驶证、驾驶证、身份证、保险单。

很多人撞了人之后不想承担责任，保险不全的还可能耍赖。**这个窗口期非常宝贵。**

---

### 5. 交警队和解：签字前必须确认一件事

治疗结束后，很多人想找交警队调解。

**但这不是必经程序，也不是交警的法定义务。**

双方必须共同向交警部门申请，任何一方不同意，交警都无权强制调解。

如果你选择和解，有一条铁律：

**拿到对方的全部赔偿款之后，再签字。**

为什么特别强调？

因为交警队的和解文书**没有法律强制执行力**。对方签完字转头不付钱，法院无法强制执行，交警也不会替你追。你只能重新起诉，白白耽误时间。

---

## 二、住院期间的六个重要提醒

### 1. 坦然面对：你着急也没用

刚住院的时候，很多人接受不了事实，见谁都不搭理。

但车祸已经发生了，这事儿不是你能控制的。

比起那些高位截瘫、植物人状态的受害人，很多伤者其实已经算幸运。

伤筋动骨100天，这是医学规律。你急，它也好不了。

**现在最重要的事：安心养伤。**

索赔的事，等伤好了再说。专业的事，交给专业的人来处理。

---

### 2. 和医生搞好关系：直接关系到你能赔多少

出院的时候，**出院记录**和**疾病诊断证明书**是主治医生写的。

上面写什么，直接决定保险公司的赔偿范围。

- 没写"加强营养" → 营养费主张不了
- 没写"需要陪护" → 护理费主张不了
- 没写"后续康复" → 后续康复费主张不了

**医生没写，保险公司就不认。你拿什么去主张？**

所以住院期间，和医生护士处理好关系，绝对必要。

这不是让你送礼走后门，而是：

- 态度上尊重医生配合治疗
- 有合理需求及时沟通
- 不要把情绪发泄在医护人员身上

---

### 3. 一个真实的反面案例

有位受害人，住院期间对医生很不满意，没事就投诉医生。

医院为了照顾他，给他换了三个医生。

结果所有人都被他投诉过。

出院时，医生写的诊断证明是：**"腰椎退行性病变及腰间盘突出手术治疗"**。

注意，不是"车祸导致"。

所有鉴定机构都拒绝给他做鉴定——**因为无法证明伤情与外伤有关，属于自身原发性疾病，评不上伤残等级。**

他去卫生局投诉医院，医院回复：腰椎退行性病变包括多种类型，也包括交通事故引起的病变。医院欢迎做医疗事故鉴定。

鉴定机构和医院是两回事，医院开了证明就算完成任务。**最后受损失的是谁？**

是这个受害人自己。

---

### 4. 伤好了再出院：别急着省那点住院费

有些受害人想着早点出院省点钱。

这是聪明反被聪明误。

**住院时间长短，直接影响赔偿。**

住院期间的费用是合理赔偿范围之内，但出院之后在家里躺着的损失，往往很难主张。

而且没完全康复就出院，万一留下后遗症，后期治疗费更难索赔。

听医生的，好了再出院。

---

### 5. 出院时必须复印的资料

出院当天，这些资料必须复印留存：

| 资料名称 | 用途 |
|---------|------|
| 出院记录 | 证明住院天数和伤情 |
| 疾病诊断证明书 | 主张各项赔偿的依据 |
| 住院费用清单 | 主张医疗费的依据 |
| 病历资料 | 完整治疗过程记录 |
| CT / X光片及报告 | 证明伤情程度 |
| 护理证明 | 主张护理费 |
| 营养证明 | 主张营养费 |

**全部复印，一份不够，要两到三份。**

理赔的时候要跑保险公司、鉴定机构、法院，资料不够用会非常被动。

---

### 6. 出院后的注意事项

- 按医嘱继续休养，不要逞强
- 定期复查，保留好所有复查记录
- 康复期内如果有新症状，及时就医并记录
- 所有票据、出院小结、检查报告分类保存

---

## 写在最后

交通事故理赔，是一个系统性工程。

从交警部门的责任认定，到住院期间的病历书写，再到出院后的鉴定和索赔，每一步都有规则，每一步也都有"坑"。

**不是伤好了就自然能拿到赔偿。中间少走一步，可能就少拿几万。**

把专业的事交给专业的人，不是认输，是聪明。
