# AI合规红线32条细读：哪些能用AI做，哪些必须人亲自做

上周金融监管总局发了一个重磅文件：《关于银行业保险业人工智能安全开发应用的指导意见》。

32条细则，七大板块，从治理架构到风险控制，全部覆盖。

很多代理人看到"32条""指导意见"这种词就划走了，觉得跟自己没关系。

**但说实话，这份文件和你的关系，比你想的要大。**

因为你在用的AI工具——无论是ChatGPT、Claude、文心一言，还是公司内部的AI助手——都在这个文件的管辖范围内。

这篇不抄法规，只说对你真正有用的东西。

## AI在保险业能做什么——监管认可的方向

先说好的。

这份文件不是来"管死"AI的。相反，它明确鼓励金融机构在以下方向推进AI应用：

- **客户服务智能化**：智能问答、保单查询、理赔指引
- **核保理赔辅助**：AI辅助审核、风险预警、反欺诈识别
- **精准营销**：客户画像分析、需求匹配推荐
- **运营提效**：文档处理、流程自动化、知识管理

大白话：**你用AI做效率工具、做辅助决策、做信息处理——监管不但不拦，还鼓励。**

平安人寿已经做到93%的理赔案件在60秒内自动审核完成，最快8秒。太保的AI员工"灵析"嵌入理赔审核环节后，人工录入字段占比降到10%以内。

这些数据说明一件事：AI在保险业的渗透是不可逆的。

**监管要管住的，从来不是"你用不用AI"，而是"你怎么用AI"。**

## AI不能做什么——必须人亲自做的红线

这才是你需要认真看清楚的部分。

### 红线一：高风险决策必须有"人在回路"

原文要点：

> 对于可解释性不足的人工智能技术，仅能作为辅助工具使用。高风险环节应建立人工监督和干预机制。

翻译：**AI可以给你建议，但最终决定必须你来做。**

- AI分析出"这个客户可能带病投保" → 但不能由AI直接拒保
- AI识别出"这份理赔可能有欺诈风险" → 但不能由AI直接拒赔
- AI建议"这个客户适合这款产品" → 但不能由AI直接推送投保链接

所有涉及客户利益的决策，必须有人的确认环节。

**"AI说可以"≠"你可以这么做"。**

### 红线二：AI生成的内容不能直接当对外材料用

原文要点：

> 生成式人工智能服务需实施准入管理，生成内容需显著标识。

你让AI帮你写了一段产品介绍、一个朋友圈文案、一篇公众号文章——

**不能直接复制粘贴就发出去。**

必须：
1. 人工审核内容的准确性（AI会产生幻觉）
2. 检查是否涉及夸大/诱导/违规表述
3. 标注内容由AI辅助生成（如果需要）

### 红线三：客户隐私数据不能随意喂给AI

> 严格落实网络安全、数据安全与个人信息保护。

这条最简单的标准：**不要让AI知道你的客户叫什么、电话多少、得了什么病。**

- 在ChatGPT里输入"我的客户张XX得了甲状腺癌，建议什么方案" → 违规
- 把客户名单上传到AI工具做分析 → 违规
- 用AI整理理赔材料但不脱敏 → 违规

**原则：任何能识别到具体个人的数据，都不应该进入公共AI服务。**

### 红线四：AI不能替代你的专业判断

> 金融机构需建立人工智能全生命周期管理体系，确保模型稳健性、透明度、可解释性。

这条比较技术，但对你的意思是：

**AI模型可能会出错，而你是最后一道防线。**

比如AI告诉你"这个客户的拒赔概率很低"——但如果条款确实不保这个病，你得自己看出来。AI看不到条款的细微差别，它只是在做概率计算。

| AI能做的事 | AI不能做的事 |
|-----------|------------|
| 快速整理条款要点 | 判断条款在实际情况中的适用性 |
| 生成产品对比初稿 | 给出最终推荐建议 |
| 分析理赔审核趋势 | 做出拒赔/通融赔付的决策 |
| 自动回复常见问题 | 回答需要专业判断的个性化问题 |
| 辅助计算保费/收益 | 保证计算结果100%准确（需人工复核） |

## 代理人日常用AI的3条安全准则

### 准则一：把AI当"实习生"，不当"专家"

你交给实习生的事：查资料、整理数据、出初稿、做对比表。

你给完指令后：必须检查一遍，改一遍，确认没问题再用。

**AI就是这个实习生。它不会犯错——如果错也是你教得不对？这种想法很危险。它会犯错，而且错得理直气壮。你必须检查。**

### 准则二：客户数据不进门

用AI处理任何客户相关信息前，先问自己三个问题：

1. **这里有没有客户的名字/身份证/电话？** → 有就脱敏
2. **有没有具体的病历/诊断/理赔金额？** → 有就概括化
3. **这个信息能不能让我妈看到？** → 不能就别输进去

一个简单的替代方案：把"I的朋友得了甲状腺癌"这种真实但不指向具体人的描述，作为输入。

### 准则三：AI产出的事，你来负责

不管你用AI做了什么——写了文案、做了分析、出了方案——

**在客户和监管眼里，这就是你做的。**

《指导意见》的第一条核心原则就是"谁使用谁负责"。

你和一个不用AI的代理人面对同样的合规要求。AI不会替你背锅。

## 总结一下

32条细则看起来多，穿上筛子看，就三件事：

**第一，AI帮你做事可以，但帮不了你担责任。**
**第二，客户数据是你最值钱的资产，也是你最大的合规风险。**
**第三，监管不是要限制你用AI，是要你别把"人的判断"外包给AI。**

而这恰恰是代理人最不可替代的价值——

AI记住了一万条条款，但不知道这个客户家的孩子刚出生、老人刚住院、自己刚失业。

**你知道。因为你见过他。这才是"人"在做保险。**
