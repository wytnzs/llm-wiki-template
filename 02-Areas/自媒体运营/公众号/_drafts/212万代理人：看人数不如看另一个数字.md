# 212万代理人：看人数不如看另一个数字

2026年一季度偿付能力报告出来了。

60家人身险公司，代理人合计2,126,294人。算上还没披露的，全行业大概220万。

活跃的可能更少。

我盯着这个数字看了很久。不是因为它在降——降了这么多年，确实不意外了。

让我停下来的，是另一个数。

**五大上市险企一季度代理人增加了将近4万。**

一边是总量在缩。一边是头部在涨。

这才是真正值得说的事。

## 头部在涨，涨的不是人头

先看总量格局。

| 公司 | 代理人数量 |
|------|-----------|
| 中国人寿 | 644,752 |
| 平安人寿 | 332,770 |
| 泰康人寿 | 275,760 |
| 太保人寿 | 190,000 |
| 太平人寿 | 182,728 |

前五家合计约137万人。一个季度净增近4万。

在行业整体缩量的背景下，这个增长是什么信号？

不是"人海战术又回来了"。

是"吸纳优才"。

我认识几个代理人，去年还在犹豫要不要走。今年跟我说：公司开始给专业培训了，AI工具也配上了，客户沟通效率比以前高很多。

以前保险公司比的是谁家人多。

现在比的是——同样100个人，谁家产出高。

## 一个季度，五分之一的人离开

再看脱落率。

东吴人寿一季度21.67%。

什么概念？每5个人里，3个月走掉1个多。

中美联泰大都会16.83%。交银人寿16.05%。

这些数字背后，是一件事——传统销售模式撑不住了。

监管严了。客户精了。只靠自保件、靠人情单的人，正在被市场清出去。

我做过内勤。见过太多代理人一年就卖两三单——还都是卖给亲戚。那种模式，经济上行的时候勉强能活。现在不行了。

不是行业不要人了。

是不要"只会卖产品"的人了。

## 人均保费的差距，才是真正的差距

有一个数据，比总量和脱落率都重要。

| 公司 | 一季度人均保费 |
|------|---------------|
| 友邦人寿 | 11.25万元 |
| 瑞泰人寿 | 10.82万元 |
| 北京人寿 | 10.71万元 |
| 交银人寿 | 10.21万元 |
| 阳光人寿 | 约2.64万元 |
| 泰康人寿 | 约1.59万元 |

友邦代理人有4.7万。不靠人海，靠单兵作战。

泰康代理人有27.6万。但人均保费不到友邦的七分之一。

差距不在人数。

在每个人能创造的产出。

我以前说过：保险代理人最值钱的能力不是"会说"，是"会想"。会用工具分析客户需求，会把复杂产品讲简单，能在客户出事的时候帮他把钱拿到。

**这些能力，AI刚好能放大。**

一个会用AI分析客户需求的代理人，和一个靠话术攻单的代理人，效率差多少？我的经验是——至少三到五倍。

## 留下的人，都在换活法

行业从900万降到220万。还会继续降。

但留下来的人，不是最能熬的。是最先想明白的。

想明白什么？

客户买的不是一张保单。是一个承诺——出了事，钱能赔下来。

你能不能帮客户把保险搞明白、把钱拿到手。

这才是"专业"两个字真正的意思。

AI不是来替代代理人的。是来帮那些已经想明白的人，跑得更快。
