# 保险代理人AI助手 · 方案梳理

> 基于产品讨论的要点整理，包含技术评估、架构选型、商业模式建议

---

## 一、原始方案的可行性评估

### 技术路径：可行

核心技术组合（飞书机器人 + 函数计算 + DeepSeek + 多维表格）逻辑成立，链路跑通过类似实现（Cloudflare Worker 那套），换平台只是部署细节差异。

### 三个需要重视的问题

**1. 多租户架构是最大的坑**

文档中"OpenID ↔ 飞书表格Token映射"看似简单，实际复杂度很高：

- 用户需要自己创建飞书多维表格并授权给机器人，这一步对0基础代理人就能劝退一半人
- 若由开发者帮助创建，则需要 OAuth 授权流程，变成中型工程
- "OpenID 映射存在本地 JSON 文件"在 Serverless 环境中行不通，冷启动后文件状态丢失，需换为云数据库

**2. 数据自有与多租户管理存在矛盾**

- 完全去中心化 → 无法实现自动提醒（生日、跟进提醒需要定时任务，函数计算默认不带）
- 定时任务需要遍历所有用户表格，权限管理复杂

**3. 买断定价与持续维护成本有矛盾**

- ¥199 买断，但有 20 人天开发量 + 持续维护 + API 成本
- 飞书 API、DeepSeek 接口变动时需升级，但买断用户不会续费
- 建议"1年更新"明确化，第二年收续费

## 二、核心设计方向调整

### 核心原则

**不做 SaaS 多租户，做一次性卖断**

- 系统完全构建在用户自己的飞书里
- 卖的是"搭建方案"，交付后与开发者无关
- 数据 100% 归用户，开发者不托管任何东西

### 交付物构成

```
你卖的东西：
├── 飞书多维表格模板（客户表 + 跟进记录表）
├── 机器人配置包（代码 + 配置教程）
├── 一份图文安装说明
└── 可选：1对1 远程代部署服务
```

## 三、技术方案选型

### 为什么不用 Cloudflare Worker

国内访问需要梯子，代理人用不了，排除。

### 国内可替代方案对比

| 方案 | 优点 | 缺点 |
|------|------|------|
| **阿里云函数计算 FC** ✅ 推荐 | 免费额度大、中文文档全、国内稳定 | 需实名注册阿里云 |
| 腾讯云 SCF | 微信生态熟悉 | 操作界面略复杂 |
| 华为云 FunctionGraph | 企业用户多 | 文档没阿里好 |
| 飞书自建应用 + 内网穿透 | 完全本地 | 需电脑一直开着，不现实 |

### 推荐方案：阿里云函数计算 FC

**逻辑与 Cloudflare Worker 完全相同，只是换了平台：**

```
用户手机
  → 飞书 APP
  → 飞书机器人
  → 阿里云函数计算（用户自己账号）
  → DeepSeek API
  → 用户自己的飞书多维表格
```

**成本**：每月前 100 万次调用免费，单个代理人用量约每月几千次，实际费用 ≈ ¥0

**用户操作流程**：

```
1. 注册阿里云（实名认证）
2. 开通函数计算 FC
3. 上传开发者提供的代码包
4. 填入自己的飞书 AppID + DeepSeek Key
5. 绑定飞书机器人回调地址
6. 开始使用
```

## 四、"用户自己部署"的现实问题

### 代理人技术水平真实分布

```
能独立部署云函数的           ── 约 5%
看教程能磕磕绊绊完成的       ── 约 15%
中途卡住放弃的               ── 约 50%
看到"实名认证"就关掉的       ── 约 30%
```

### 核心矛盾

| 期望 | 现实 |
|------|------|
| 一次性卖断，不托管系统 | ✅ 可以做到 |
| 用户数据自有 | ✅ 可以做到 |
| 售后压力小 | ❌ 用户自己部署 → 售后最多 |
| 操作简单 | ❌ 云函数部署对代理人不简单 |

---

## 五、最终建议：代部署模式

### 为什么代部署是最优解

1. **技术复杂度问题由你解决**，用户只管用
2. **系统跑在用户自己账号里**，你不托管数据，符合数据自有原则
3. **你配置一次只需 30 分钟**，前期用户少时完全可控
4. **同时是你 IP 价值的体现**：帮代理人用上 AI 工具，本身就是"保险其实不难"IP 的内容素材

### 执行流程

```
用户付款
  → 加你微信
  → 你远程配置（约 30 分钟）
  → 用户直接开始使用
  → 系统完全跑在用户自己飞书 + 阿里云账号里
```

### 建议定价

| 产品 | 价格 | 内容 |
|------|------|------|
| 教程版 | ¥99 | 图文教程，自己部署（适合有技术基础的） |
| 代部署版 | ¥299-399 | 你远程帮装好，用户直接用 |
| 尊享版 | ¥599 | 代部署 + 6个月技术支持 |
