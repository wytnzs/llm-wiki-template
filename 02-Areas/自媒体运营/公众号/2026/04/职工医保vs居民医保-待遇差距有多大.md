# 同一个手术，职工花1万居民花5万——石家庄医保差距真相

同一个城市，同一个医院，同一台手术。

因为身上贴的标签不一样——一个"职工"，一个"居民"——最后自己掏的钱能差好几万。

这不是段子。这是石家庄2026年正在发生的事。

## 先看差距有多大

我用一张表把两套体系摊开比。

| 比较维度 | 职工医保 | 居民医保 |
|---------|---------|---------|
| 2025年个人缴费 | 工资的6.5%（约3870-7160元/年） | 400元/年 |
| 财政补贴 | 无（单位承担大头） | 不低于670元/人/年 |
| 个人账户 | 有（可用于药店+共济） | 无 |
| 门诊年度报销上限 | 最高6000元 | 400元 |
| 门诊起付线 | 100元 | 0元（2026年新规） |
| 门诊报销比例 | 60%-70% | 最高95%（基层） |
| 住院年度封顶线 | **75万元** | 约30万元（含大病） |
| 退休待遇 | 缴满年限终身免费 | 需逐年缴费，无退休概念 |

光看缴费，居民医保便宜得离谱——一年400块，一顿饭钱。

但看门诊限额——职工最高6000，居民最高400。

400块。一年。够看什么？

一次感冒+一次慢性病复查，基本就没了。

## 为什么差距这么大——底层逻辑不一样

两种医保的名字已经说明了它们的设计逻辑。

**职工医保**，全称"城镇职工基本医疗保险"。"职工"两个字意味着：你有工作，有人给你交大头。单位和个人共同缴费，强制参保。本质上是一种**劳动福利**。

**居民医保**，全称"城乡居民基本医疗保险"。"居民"两个字意味着：你没工作，或者没有固定工作。个人交一点，财政补大头。本质上是一种**社会救助**。

一个定位是"给有工作的人发福利"，一个定位是"给所有人兜个底"。

定位不一样，差距就是天生的。

## 门诊——居民医保最真实的短板

2026年3月前，居民医保的门诊限额是200元。

200元。一年。这就是一个普通感冒+一盒阿奇霉素的钱。

2026年3月新政后提到了400元，报销比例提到最高95%，还取消了起付线。改善很大，但和职工医保的6000元比起来——15倍的差距。

在职工那边，门诊共济改革后，起付线降到100元，退休人员限额5000-6000元，还带着个人账户，药店买药也能报。

这意味着什么？

一个职工感冒了去药店刷个感冒药，走个人账户。一个居民感冒了得去医院排队挂号，才能用门诊统筹——因为居民没有个人账户。

用户体验完全不同。

## 住院——最怕的不是生病，是大病

住院保障上，差距更吓人。

职工医保的年度封顶线在2026年提高到了75万元。居民医保呢？含大病保险在内，大约30万左右。

2.5倍的差距。

说句实话，30万对大多数家庭来说已经不少了。石家庄2025年城镇居民人均可支配收入大约4.5万，30万相当于将近7年的收入。

问题在哪？在极端风险。

癌症靶向治疗、器官移植、重大手术——这些场景的费用可以在几个月内烧掉30万。到了这个量级，职工医保的75万封顶线才有真正的缓冲空间。

## 退休——一道分水岭

职工医保有"累计缴费年限"的概念。男的缴满30年、女的缴满25年（各地略有差异），退休后不用再交一分钱，终身享受医保待遇。

居民医保没有"退休"的概念。今年交了今年有，明年不交明年没。不存在"我交够了就可以歇着"这回事。

一个60岁退休的职工和一个60岁的居民，在医疗上的安全感是完全不同的。

## 灵活就业人员——最纠结的一群人

石家庄2025年把灵活就业人员的缴费比例从8%降到了6.5%。

按2021年全市平均工资71463元算，一年缴约4645元。

4645元 vs 400元——差了11倍。

但换来的东西完全不同：职工医保的个人账户、门诊6000额度、住院75万封顶、退休终身待遇。

对灵活就业人员来说，这不是"贵不贵"的问题，是"值不值"的问题。如果长期干自由职业，多交的4000块钱分摊到12个月，每月多300多块——换来的是一个完全不同的保障层级。

## 所以到底怎么选？

如果你或你的客户正在纠结，我的建议是：

**有工作的**——职工医保是强制的，没得选，但也别抱怨。它是你性价比最高的保障，没有之一。

**自由职业/灵活就业的**——优先考虑职工医保。石家庄缴费比例降到6.5%后，负担已经轻了不少。每月多交几百块，换来的是退休终身保障。

**全职主妇/主夫/学生的**——居民医保是唯一选项。别嫌它保障弱，它最大的价值是低成本维持一个"医保身份"。在这个身份之上，用商业医疗险补齐短板。

**给所有人的一句话**：

职工医保和居民医保不是"哪个更好"的问题，它们服务的是完全不同的群体。

居民医保是安全网——保你不掉下去。
职工医保是护城河——让你过得更好。

认识差距，但不否定差距。然后在自己的位置上，做最优的补充配置。

**最糟糕的选择，不是选了居民医保——而是以为自己有保障，就什么都不用再准备了。**
