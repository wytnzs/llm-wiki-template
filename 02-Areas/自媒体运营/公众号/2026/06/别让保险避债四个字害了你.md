# 别让"保险避债"四个字害了你

去年有个代理人找我，说自己卖保险五年，跟每个客户都说"保险能避债"。

结果上个月一个客户收到法院传票，他名下三份保单被冻结了。

客户打电话来，语气很冲："你不是说保险谁也拿不走吗？"

代理人翻遍了合同，没找到"避债"两个字。后来查了一堆资料才明白——

**他卖了五年保险，自己根本没搞清楚保险和债务的关系。**


## 一、"保险避债"这句话是怎么来的

这句话不是从法律条文里来的。

它的源头大概是十年前，一些保险销售培训里讲的一个观点：

> "《保险法》规定，任何单位和个人不得限制被保险人或者受益人取得保险金的权利。"

原文是真的。出自《保险法》第二十三条。

但原文说的根本不是什么"保险能避债"——它说的是**已经出险、确定要赔的保险金**，任何单位不能截留。

这句话在传播中被越传越变形：

> "保险金不能被执行"→ "保单不能被执行"→ "保险能避债"→ "欠债不用还"

**从一条正确的法律条文，推导出了一个完全错误的结论。**


## 二、真相一：最高法2021年就一锤定音了

2021年，最高法做了一个裁定。案号（2021）最高法执监35号。

案情不复杂：一个人欠了钱，法院查到他在某保险公司有几份保单，有现金价值。法院要求保险公司协助执行——强制解除合同，把现金价值交出来。

被执行人不服，一路申诉到最高法。

**最高法的结论：保单现金价值是投保人的财产，可以被执行。**

核心论证逻辑有三条：

1. 人寿保单有现金价值——现金价值具有**财产属性**，不是"人身权益"
2. 投保人可以随时单方面解除合同拿回现金价值——这个权利本质上是财产权
3. 被执行人欠债不还，又不主动解除合同拿钱还债——**法院有权替代他执行**

这个裁定至今没有推翻过。

**它的意思很清楚：你名下的储蓄型保险、分红型保险、万能型保险——只要有现金价值，在法院眼里和银行存款本质上是一回事。**


## 三、真相二：不是所有保单都"安全"

到这里你会问：那保险还有没有用？买的保单是不是白买了？

**不是。关键看你买的是什么。**

| 保单类型 | 有没有现金价值 | 法院能不能执行 | 典型案例 |
|---------|--------------|--------------|---------|
| 一年期医疗险 | 无 | 一般不执行 | 没有现金价值可执行 |
| 一年期意外险 | 无 | 一般不执行 | 同上 |
| 消费型重疾险（定期） | 很少/无 | 一般不执行 | 保障功能为主 |
| 终身重疾险（带身故） | 有 | **可能执行** | 上海高院：重疾险一般不作扣划 |
| 终身寿险 | 有 | **可能执行** | 广州白云法院2026年案例 |
| 年金险 | 有 | **可能执行** | 宁夏平罗法院2025年案例 |
| 分红险/万能险 | 有 | **可能执行** | 湖北公安检察院监督案 |
| 投资连结险 | 有 | **可能执行** | 浙江高院2015年有规定 |

**但注意：上海高院2021年会议纪要明确了一个重要原则——保障功能强的产品，法院一般不作扣划。**

也就是说：你买了一份终身重疾险，因为欠债被法院执行——法院会先评估这份保单的保障功能。如果被执行人为本人或家属购买，且保单的重疾保障对其家庭有重要意义，法院**会审慎处理**，不一定直接扣划。

那真正"危险"的是哪几类？

**分红型、万能型、年金型、投资连结型**——这些本质上就是理财产品，披了一张保险的皮。法院拿它们开刀，几乎没有心理障碍。

**一句话：保障归保障，理财归理财。想靠理财型保单"避债"，法院不认。**


## 四、真相三：法院的执行链条比你想象的完整

法院不是突然就把你的保单"拿走"的。整个过程至少四步：

**第一步：查询锁定。** 法院发函到各家保险公司，查询被执行人名下保单。现在保险行业已经建立起集中的保单信息查询系统——你买了什么、在哪家买的、交了多少保费、现金价值多少，一台电脑就能查到。

**第二步：冻结通知。** 查到后有现金价值的保单，法院发《协助执行通知书》给保险公司，冻结该保单的现金价值。冻结期间不能退保、不能贷款、不能变更。

**第三步：赎买期。** 法院会给被执行**不少于15天**的时间来处理。这期间，被执行人可以选择自己凑钱还债，保全保单。也可以由被保险人或受益人拿钱出来"赎买"——这就是《保险法》说的**介入权**。

**第四步：强制解除。** 15天到期还没动作，法院直接发函给保险公司：强制解除这份保险合同，把现金价值划入法院账户。

**2026年广州白云法院有个真实的案例。**

被执行人黄某欠了钱，法院查到他名下有平安人寿的保单，现金价值近12万。法院给了15天赎买期——他没还钱。法院就发了强制解除通知，保险公司把11.9万现金价值划到了法院账户。

**但这个故事还有一个尾巴。**

这份保单的被保险人是黄某的三个未成年子女。孩子的外公外婆不干了——他们要求变更自己为投保人，补缴了等额的现金价值给法院，把保单"救"了回来。

法院同意了。因为《保险法解释三》第17条明确给了被保险人/受益人这个权利：**你把钱替投保人还了，保单可以继续有效。**

这个案例说明两件事：

**第一，法院确实会强制解除保单——不是吓唬人的。**

**第二，法律也给了"抢救"的方法——但不是投保人自己用。**


## 五、给代理人的5套实操方法

下面这部分是**代理人能直接用的**。每个方法都有具体场景、操作步骤、话术参考。

### 方法一：4种客户画像，4套保单架构方案

不同客户情况，保单架构完全不同。下面是具体的判断方法和操作步骤。

**场景A：客户给自己买保障型产品（重疾/医疗/意外险）**

→ 怎么搭：投保人写客户自己，没问题。

→ 为什么：这类保单现金价值很低甚至没有，保障功能强，法院根据比例原则一般不执行。上海高院2021年会议纪要明确写了这一点。

→ 但有一点必须做：受益人指定具体的人，不能写"法定"。

→ 话术参考："张姐，你这份重疾险买自己名下完全OK，保障型的法院碰不了。但受益人那栏我帮你填上你孩子的名字——万一用到，钱直接到孩子手上，不走遗产程序。"

**场景B：客户给自己买储蓄型产品（年金/增额终身寿/分红险）**

→ 怎么搭：投保人写客户的父母或配偶，客户做被保险人或受益人。

→ 为什么：现金价值属于投保人。投保人不是客户，客户的债主就动不了这份保单的现金价值。

→ 操作细节：
1. 保费来源：客户把钱转给父母/配偶，由父母/配偶的账户支付保费
2. 转账保留：客户转账时备注"赡养费"或"家庭生活开支"
3. 投保人签投保单：由父母/配偶亲笔签名，客户不能代签

→ 话术参考："王总，你要买这份年金险做养老补充，对吧？那我建议投保人写你家阿姨（妻子）。钱你先转给她，她来交保费。这样保单在法律上就是她的资产——你那边生意上的债务风险，不会影响到这张单子。"

**场景C：客户给子女买保险**

→ 怎么搭：投保人写谁？三种情况对号入座。

| 客户情况 | 投保人写谁 | 为什么 | 注意 |
|---------|-----------|-------|------|
| 客户收入稳定，债务风险低 | 客户自己 | 操作简单 | 受益人写子女 |
| 客户自己做生意，有债务风险 | 客户父母（孩子的爷爷奶奶/外公外婆） | 隔代投保，现金价值不在客户名下 | 部分产品对投保人年龄有限制 |
| 客户已经欠债 | 配偶（或更复杂安排） | 隔离客户债务 | ⚠️ 债务人期间变更投保人可能被认定为恶意转移 |

→ 操作细节（隔代投保）：爷爷奶奶做投保人，客户做受益人之一（或不做受益人，只让子女做受益人）。保费由爷爷奶奶的账户出，客户给爷爷奶奶转账，备注"赡养费"。

**场景D：夫妻互保（A给B买，B给A买）**

→ 这样的结构下，如果A欠债了：
- A做投保人的保单（给B买的）→ 现金价值属于A → **会被执行**
- B做投保人的保单（给A买的）→ 现金价值属于B → **不会被执行**

→ 结论：夫妻互保不能完全隔离债务，只能隔离一半。真正安全的做法是让没有债务风险的一方做投保人，给双方都买好。

### 方法二：受益人安排——3个隐藏的坑

**第一个坑：写了"法定"或"遗产"。**

这是最常见的问题。投保时随手写"法定"，结果是：保险金变成遗产 → 先还债 → 剩下的才分给家人。保险白买了。

→ 操作：投保单上**必须写具体姓名+身份证号+关系+受益比例**。

**第二个坑：只写了一个受益人，没有备选。**

假设受益人写的是配偶。如果配偶先于被保险人身故，保险金自动变遗产 → 被执行。

→ 操作：设置多顺位受益人。
- 第一顺位：配偶 50%，长子 50%
- 第二顺位：客户父母 100%

这样不管什么情况，钱都不进遗产。

**第三个坑：受益人写的是未成年人。**

如果保险金赔付时受益人才5岁。保险公司通常不会直接把大额保险金支付给5岁孩子，而是由法定监护人代管。这时候钱在监护人名下，可能被监护人的债主执行。

→ 操作：可以考虑设置保险金信托，或者指定成年亲属作为收益人之一。

### 方法三：新单"债务隔离架构"的三层搭法

代理人给客户做方案时，可以按这个三层架构来设计：

**第一层：保障型保单——客户自己名下。**
重疾险、医疗险、意外险。现金价值低或无，法院一般不执行。投保人写客户自己。

**第二层：储蓄型保单——隔离在家庭成员名下。**
年金险、增额终身寿险。投保人写没有债务风险的亲属（老人、配偶）。
保费路径：客户 → 转账亲属（备注赡养费/生活费）→ 亲属账户 → 交保费。

**第三层：大额资产传承——搭配保险金信托。**
年缴保费50万以上的客户，建议搭配信托。投保人写信托公司，保单资产与客户个人债务完全隔离。需要法律专业人士配合。

### 方法四：现有保单"债务风险体检"——代理人可以直接用的排查清单

上门见客户时，拿这个清单过一遍，5分钟就能帮客户找出风险点：

```
□ 客户名下的年金险/分红险/万能险，投保人写的是谁？
   → 写自己 → 有风险。建议尽早评估是否变更投保人
   
□ 客户名下所有保单的受益人，写了具体姓名还是"法定"？
   → "法定" → 有风险。建议补充指定受益人
   
□ 客户给子女买的保险，投保人写的是谁？
   → 写客户自己且客户有债务风险 → 建议改隔代投保
   
□ 客户有没有大额保单贷款？
   → 有 → 需要确认贷款用途和还款计划
   
□ 客户有没有更换过投保人？
   → 有 → 需要确认变更时间是否在债务发生前（恶意转移的认定时效）
```

### 方法五：客户已经面临债务风险——三步应对法

如果你的客户已经被债权人起诉或被执行了，按这三步走：

**第一步：确认保单现状。**
查询自己名下所有保单的投保人、被保险人、受益人、现金价值信息。不要猜测，直接看合同。

**第二步：判断保单类别风险等级。**
- 高优先保护（重疾/医疗/意外）：法院一般不执行
- 中等风险（终身寿险/重疾带身故）：视法院裁量
- 高风险（年金/分红/万能/投连）：现金价值高，最容易被执行

**第三步：用"介入权"抢救。**
如果保单已经被冻结，让其他有资格的人（被保险人、受益人、或者他们的近亲属）向法院申请行使"介入权"——即拿出等额的现金价值替投保人还债，保单变更投保人，继续有效。

**但注意：**
- 客户**已经欠债了才去买保险**，或者**明知要欠债了突击买保险**——这在法律上叫"恶意转移财产"，法院可以撤销。最早要在债务风险出现前**半年到一年以上**完成保单配置。
- 客户**已经欠债了才变更投保人**——同样可能被认定为恶意转移。

## 对代理人：这5句话直接跟客户说

这几句话不绕、不虚，客户听完能记住：

**第一，保障型的保险（重疾/医疗/意外）法院一般不碰——让你客户放心买。**

**第二，储蓄型的保险（年金/分红/万能）法院会执行——投保人别写自己。**

**第三，受益人一定要写具体的人，写"法定"等于白买。**

**第四，债务来了才买保险——来不及了，法院会查时间。**

**第五，保单被冻结还有救——让家人行使介入权赎回来。**

保险最大的功能是保障——不是帮你还债。别被"保险避债"四个字害了自己，也别用它害了客户。
