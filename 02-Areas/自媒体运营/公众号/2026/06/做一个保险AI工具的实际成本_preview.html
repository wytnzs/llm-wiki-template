经常有人问我：你做的那些工具，花了多少钱？

拒赔维权分析、伤残鉴定预估、家庭财务分析——听起来像是要一个团队干半年。实际上，三个工具加在一起，花了几千块。时间加起来不到三个月。一个人。

这篇文章把成本全拆开。如果你想自己做工具，可以当预算参考。

总计：约 9 周，1700 元。

钱花在哪了

大模型 API 调用费（最大头）：DeepSeek API 开发阶段反复测试约 1000 块。上线后一个请求 0.3-0.8 元，用户量没起来之前一个月几十块。

大模型 API 是最大的开支。但选对模型，成本可控。DeepSeek 的性价比目前没有对手。

服务器和域名：Cloudflare Worker 免费额度够用。域名一年几十块。基本可以忽略。

数据采购：伤残鉴定工具买《人体损伤致残程度分级》完整数据和各地赔偿标准表，花了 500 块。其他两个工具用的公开数据——免费。

设计和其他：没花钱。工具界面就是表单+结果展示，不需要设计。

文档

做工具最难的不是"做出来"。是"做到能用"。

第一阶段：搭原型（30%）——Claude Code 2-3 天搭出能跑的原型。不会写代码也能做。
第二阶段：调效果（40%）——最花时间。反复改提示词框架、补数据、跑测试用例。没有捷径。
第三阶段：找人试（20%）——他们的吐槽比点赞值钱一百倍。
第四阶段：文档（10%）——再好的工具，没人知道怎么用，等于没做。

你的第一个工具，怎么开始

第一步：选一个你每周至少遇到三次的问题。别想做"万能工具"。
第二步：先用现有 AI 试。看看能不能答。能答但不够好——说明缺的是场景框架和数据。这就是你的切入点。
第三步：搭一个最简原型。让 Claude Code 帮你写。
第四步：找 5 个人试。别找夸你的人。

第一个工具的目标不是"好用"。是"能跑"。能跑之后，迭代才是真正的开发。

值不值

一个工具帮 100 个人，一个人省 30 分钟——就是 3000 分钟。我花 9 周换 3000 分钟，值。而且工具是资产。你做完了，它就是你的。明年还在用，后面还在用。

理赔帮一个人。工具帮一群人。这就是我为什么一直在做。
