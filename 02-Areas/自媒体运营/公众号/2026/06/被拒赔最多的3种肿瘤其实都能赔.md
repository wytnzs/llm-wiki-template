# 被拒赔最多的3种肿瘤，其实都能赔

去年有个代理人找我，说客户拿着拒赔通知书来找他，上面写着"不属于恶性肿瘤"。

客户确诊的是宫颈上皮内瘤变三级（CIN III），做完了锥切手术，花了小两万。保险公司说：这不是原位癌，不在保障范围。

代理人很无奈——他翻遍了合同条款，确实没找到"上皮内瘤变"四个字。

我说：你先别急。这三个字能不能赔，不看字面，看编码。

## 一、CIN III：名字里没有"原位癌"，但它就是

**宫颈上皮内瘤变三级**，是女性生殖系统最常见的癌前病变之一。子宫颈的上皮细胞被异型细胞取代，已经发展到上皮全层，只差穿破基底膜变成浸润癌。

从病理学上讲，CIN III = 重度非典型增生 + **原位癌**。

ICD-O-3编码：8077-2，斜杠后是**2**，代表原位癌。

**但很多保险公司不认。**

理由是合同轻症条款写的是"原位癌"，病理报告写的是"上皮内瘤变"或"高级别鳞状上皮病变"——字面不符，拒赔。

2025年7月，山东一位客户就遇到了这个情况。CIN III累腺，做完手术，向和谐健康申请理赔，对方回复：不构成赔偿条件。提出给6万"人道补偿"，要求退保。

客户没同意，找了律师。

律师引用了ICD-10、ICD-11、WHO女性生殖器官肿瘤分类第五版，证据链完整。媒体跟进报道后，4天之内，保险公司**同意按合同正常赔付**。

**为什么？因为从编码到病理，CIN III属于原位癌在法律和医学上"不存在争议"。**

包括宫颈高级别鳞状上皮病变（HSIL），在2018版宫颈癌诊疗规范中，它的编码同样是8077-2。也就是说，HSIL和CIN III一样，在肿瘤分类上被定性为原位癌。

**结论很简单：CIN III和HSIL，都应该按原位癌理赔轻症保险金。**

维权路径：
1. **病理科会诊**：请医生在报告上注明"CIN III（相当于原位癌）"
2. **固化证据**：复印全套病案，确认编码为8077-2
3. **异议申诉**：引用《保险法》第三十条，格式条款有歧义时，应作有利于被保险人的解释

## 二、NET G1：买了旧版重疾险，100%赔

神经内分泌瘤（NET G1），以前叫"类癌"——听起来像良性，实际上它有恶性特征。

早期WHO把NET归为交界性肿瘤，编码后面是/1。但到2019年，WHO已经明确：结直肠神经内分泌瘤的动态编码是**/3**——恶性肿瘤。

而保险理赔的纠纷，往往卡在时间差上。

**最典型的案例：福州一位客户同时买了三家公司重疾险。**

客户2020年查出"直肠神经内分泌瘤（G1），浸润粘膜层至粘膜下层"。他买了三家公司的重疾险：平安和人保都赔了（合计约208万），但和谐健康拒赔。

拒赔理由："病理报告写了浸润，但没写'破坏周围正常组织'，达不到合同里的恶性肿瘤标准。"

一审输了，客户上诉。

福州中院二审翻盘。判决的核心逻辑是：

**2020版重疾新规把G1级神经内分泌瘤定义为"恶性肿瘤—轻度"，反推——它在2007版合同下就是恶性肿瘤。2007版不分轻重，确诊就应全额赔付。**

这个逻辑非常有杀伤力。它意味着：**保险合同定义的滞后性，不能成为剥夺被保险人理赔权利的理由。**

**维权关键**：先看保单版本。
- 2007版合同 → 坚持要求全额赔付，援引陈浩案判例
- 2020版合同 → 接受轻症赔付（20%-30%保额），除非有淋巴结或远处转移

不管哪个版本，理赔依据都是病理报告中的"浸润粘膜下层"——这本身就是恶性肿瘤的侵袭性特征。

## 三、GIST：WHO已经把编码改成/3了，保险公司的/1已经过时

胃肠道间质瘤（GIST），是消化道最常见的间叶组织肿瘤，60%-70%发生在胃部。

它的特殊之处在于：**没有绝对的良性。**

以前医学界把GIST分成良性、交界性和恶性。但在最新的WHO消化系统肿瘤分类中，GIST已经没有"良性"这个分类了——全部归为恶性肿瘤，编码从老的8936/1（交界性）更新为**8936/3（恶性）**。

但很多保险公司还在用老编码。

有个真实的案例：一位客户做了胃部手术，病理报告诊断是"胃肠道间质瘤，潜在恶性"，医院给的ICD编码是D37.101（交界性肿瘤）和M89360/1。保险公司一看——/1，交界性，不赔。

**但这位客户后来赔了。**

他找到主治医生，说明了WHO（2019）分类标准的最新变化。医生按新标准重新出具了《住院病案首页》，把诊断修正为"胃恶性肿瘤"（C16.900）和"恶性肠胃道间质性瘤"（M89360/3）。

**编码从/1改成/3，保险公司最终赔付了。**

GIST的赔付还有一个硬证据：**靶向药。**

中高危的GIST患者术后需要长期服用伊马替尼（格列卫），这是抗肿瘤靶向药，一盒上万。能开这种药的处方，本身就是疾病严重程度的铁证。

**GIST维权路径：**
1. **锁死免疫组化**：确认CD117、DOG1阳性（GIST特异性标志）
2. **固定危险度评级**：让医生按NIH 2008改良版做危险度判定
3. **修正编码**：联系医生依据WHO最新分类重新出具病案首页
4. **递交专业申诉**：附上WHO最新分类标准，主张GIST的/3属性

## 记住三条

如果你或你的客户正遇到类似情况：

- CIN III的编码8077-2——2就是原位癌，名字不同≠不是
- 2007版合同下的NET G1应当全额赔付——已有判例支持
- WHO已经把GIST的编码改成了/3——保险公司的/1已经作废

**拒赔通知书不是终点。它是一个起点——一个让你认真审视编码和条款之间差距的起点。**
