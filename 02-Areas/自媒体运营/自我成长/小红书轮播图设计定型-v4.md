# 小红书轮播图设计定型 — 炭白奢感风格 v4

> 定型日期：2026-06-01
> 迭代过程：v2(炭黑极简) → v3 → v4(炭白奢感) ✅ 定型

## 核心风格

- **色调**：黑白灰 + 暖金 `#C79A5E` 点缀（用量 < 5%）
- **字体**：无衬线为主，Noto Serif SC 衬线用于金句/数字
- **背景**：纸白 `#F5F5F0`（浅色页）+ 极黑 `#1A1A1A`（深色页/封面）
- **卡片**：白底 + 微阴影 + 8px 圆角
- **封面**：标准居中版，几何装饰环 + 网格纹理 + 金角 + 环境光
- **动效**：淡入上滑（`fadeUp` 0.5s）

## 定型组件库

| 组件 | 适用 | 说明 |
|------|------|------|
| 信息卡 `info-card` | 全部 | 白底微阴影内容容器 |
| 2列网格 `grid-2` | 数据对比 | 带数字的并排卡片 |
| 3列图标卡 `grid-3` | 概念列举 | 带 emoji 的三列卡片 |
| 数据表格 `data-table` | 参数对比 | 带表头的表格 |
| 步骤条 `step` | 流程说明 | 圆形数字 + 文字 |
| 对比框 `compare-box` | 左右对比 | 深色/浅色两栏 |
| 高亮框 `highlight-box` | 重点强调 | 深色渐变 + 金色标签 |
| 金句引用 `quote-box` | 金句展示 | 衬线字体 + 引号装饰 |
| CTA 按钮栏 `cta-row` | 行动召唤 | 黑色/金色按钮 |
| CTA 页 `cta-slide` | 最后一页 | 微信号 + 引导文案 |

## 页面结构（10张）

1. **封面** — 大标题 + 标签 + 金句
2. **结果页** — 核心数据展示
3-4. **概念页** — 核心概念拆解
5-7. **流程页** — 步骤/对比/流程
8. **亮点/对比页** — 左右对比
9. **方法论/金句页** — 深色 + 衬线
10. **CTA 页** — 微信号 + 引导

## 淘汰方案

- **v5 数据进度条**：测试后觉得破坏了极简感，未采用
- **v5 清单列表**：适用场景有限，保留在规范中作为可选
- **v5 垂直步骤条**：替代方案，当前水平步骤条已够用
- **v6 封面变体C**：太极端，标准居中版更通用
- **v5/v6 新增组件**：大部分未采用，保持组件库精简
