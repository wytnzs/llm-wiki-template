# 保险公司理赔调查知识整理

> 来源：刘立群《保险公司如何调查》培训课件

---

## 一、需要调查的案件类型

### （一）报案阶段应发起调查的情形

以下案件在报案信息录入完成后，应立即发起调查：

1. **身故未葬者**——尸体尚未火化/安葬，需及时查勘取证
2. **残疾理赔，已届满鉴定时机者**——达到伤残鉴定时间节点
3. **正在医院住院治疗者**，包括：
   - 患重疾、心脏疾病（心功能不全II级以上）
   - 高血压病（II级以上）、糖尿病、肝硬化、慢性肾脏疾病、肾功能不全
   - 再生障碍性贫血、癫痫、系统性红斑狼疮
   - 责任范围内手术等
4. **保险事故涉及公安刑事处理，且尚未结案者**
5. **出险事故符合重大紧急案件标准者**
6. **上级核赔主管认为有必要提起调查者**

### （二）重大紧急案件定义

满足以下任一条件即属重大案件：

1. **知名人士/重要客户**：出险人在当地为知名人士或重要客户，因其特殊身份被公司、行业内、媒体及社会公众广泛关注
2. **公共灾难**：涉及多人伤亡（5名以上），主要包括火灾、爆炸、山体滑坡、塌方、中毒、海难、空难、恶性交通事故等
3. **社会影响重大**：造成被保险人出险的事故在当地有重要影响，成为社会焦点，如民事及刑事案件、政治案件

### （三）审核阶段应发起调查的情形

审核阶段的调查重点倾向于保单性质、逆选择因素及责任免除排除：

1. **身故/高残类**：
   - 保单生效或复效后二年内因疾病身故，预计超过一定金额。
   - 二年后预计超过一定金额。
   - 因意外身故，预计超过一定金额。
   - 意外伤害为保单唯一责任且导致身故/高残的案件

2. **伤残类**：预计伤残给付超过一定金额。

3. **重疾类**：
   - 保单生效或复效后一年内患重疾
   - 二年内患重疾，预计超过一定金额。
   - 二年后重疾出险预计给付超过一定金额。

4. **医疗类**：
   - 观察期后二个月内患疾病，医疗给付超过一定金额。
   - 保单生效半年内医疗给付超过一定金额。
   - 预计医疗给付超过一定金额。

5. **特殊风险控制类**：
   - 疑有道德危险
   - 有投保前之疾病或伤害
   - 有明确除外责任可能（自杀、故意行为、犯罪嫌疑等）
   - 伤势判断、治疗情况及费用明显不合理
   - 因疾病或意外伤害所致不明
   - 申请索赔次数频繁
   - 医院、被保险人、业务员有不良记录
   - 存在其他保险欺诈可能或确有必要调查

## 二、调查流程与要点

### （一）调查流程总览

```
报案 → 提起调查 → 任务分配 → 获取任务 → 调查取证 → 调查结论
```

### （二）报案调查阶段

**调查性质**：外务调查。

**调查目的**：及时把握调查有利时机，防止证据丢失或无法采集。

**调查时间要求**：
- 符合报案调查条件的，核赔人应于接报案当日在系统内提交调查
- 查勘人员须在发起调查后 **6个工作小时以内** 启动并开展调查工作

**调查项目分类**：
| 项目 | 说明 |
|------|------|
| 医疗查勘 | 围绕医疗事件开展 |
| 事故查勘 | 对非医疗事件开展现场取证 |
| 基础查勘 | 围绕出险人/投保人/受益人的生活、工作、财务等 |
| 正面查勘 | 与被保险人或受益人正面沟通会晤 |

**调查概述填写规范**：发起调查人应录入详细的调查细节要求，例如要求调查某某机关、访谈邻居、访谈某个证人等。

**禁忌**：必须向报案人详细了解并记录客户的联系电话和与调查地点、人物相关的详细信息。

### （三）审核调查阶段

**调查类型**：

1. **内务调查**——审核人自己开展的调查，包括电话沟通、核实、约见面谈、调阅档案资料等
2. **外务调查**——必须由专职查勘人员出外勤的调查，包括现场查勘、走访、医院/公安等部门调阅书证等

**调查时间要求**：符合调查的案件应立即在系统发起调查；查勘人员须在发起调查后 **8个工作小时内** 启动并开展调查工作。

**调查结论查看**：
- 调查过程记录：对同一调查对象存在多次调查的日志记录
- 机构调查结论：分属不同机构的调查任务，有多家机构是否结束的结论记录
- 赔案调查结论：同时发起的多个调查任务，是否存在结束调查的结论

**审核人提示**：应经常性主动在系统查看调查任务的查勘进展情况，以便及时审核处理赔案。

### （四）调查分配原则

1. **能力对位原则**——按专业能力分配，如疑难重大疾病分配给有医学背景的调查人员
2. **区域管理原则**——根据地理位置集中分配，提高效率
3. **工作时间原则**——根据调查人员的工作时间和工作量公平分配
4. **利益回避原则**——调查人与赔案有直接或间接利益关系的应回避

**禁忌**：应定期调整更换调查人员的管理区域，避免滋生内部人员职业道德风险。

### （五）调查取证


#### 3. 调查时限

- 普通调查案件：**2个工作日**
- 特殊或疑难案件：经请示核赔主管，可延长 **10个工作日**

#### 4. 查勘方法与途径

##### （1）现场查勘

"不到现场难于掌握第一手资料"——是核实案件真实性和发现调查线索的重要方法。

记录方式：**照相、笔录、绘图、录相、录音**

**现场照相四种方法**：

| 类型 | 内容 | 拍摄方法 |
|------|------|----------|
| 方位照相 | 反映现场在周围环境中的位置 | 直线分段拍摄法、回转连续拍摄法 |
| 概览照相 | 反映现场全貌及内部各部分联系 | 相向拍摄法、分段拍摄法 |
| 中心照相 | 反映中心部位主要物品特征 | 中心附近较高位置拍摄 |
| 细目照相 | 反映有证据意义的痕迹、物品 | 比例照相法 |

**现场查勘笔录三部分**：
- **前言**：报案时间、报案人身份、事故发生经过、查勘对象/时间/地点、现场保护情况
- **叙述事实**：周围环境、现场结构、查勘顺序/过程/方法/结果、发现的痕迹物品、异常情况
- **结尾**：受伤人员救治、尸体处理、提取物品的名称/数量/方法、查勘人员及事主签名

**制作要求**：用语准确、重点突出、内容客观，绝不能将个人主观分析推断记入笔录。

##### （2）询问

**对象**：当事人、被保险人及其亲友、保险代理人、职能部门工作人员、专家、知情人。

**方式**：
- **直接询问**：面对面交谈（询问保险金申请人、走访知情人、面晤医师/交警等）
- **间接询问**：借助电话、传真、书信（电话询问可不暴露身份和真实意图，可录音）

**询问内容**：
- 发生和发现事故的时间、地点及详细经过
- 事故发生前后看到或听到的情况
- 现场情况有无变动及原因
- 被保险人出事前后的神态、言行
- 投保前后的身体状况、家庭经济情况、家庭成员关系
- 知情人对事故性质的看法和议论

**询问技巧**：
- 事先制定详细询问计划，穷尽各种可能性
- 掌握询问主动权，有效控制询问过程
- 运用回忆方法促进被询问人回忆：**接近回忆**（时间空间接近的事物）、**相似回忆**（相似事物）、**对比回忆**（相反事物）、**关系回忆**（事物之间的联系）

**询问笔录结构**：
- **开始部分**：时间、地点、询问人姓名、被询问人情况（姓名/性别/年龄/住址/工作/联系方式）、与被保险人关系
- **陈述部分**：一问一答形式，问题严谨且符合逻辑性
- **结尾部分**：被询问人核对签名、询问人签名、翻译/监护人签名

##### （3）鉴定

**鉴定事项**：死者身份鉴定、事故原因鉴定、事故性质鉴定、伤残等级鉴定、笔迹鉴定

**鉴定种类**：

| 种类 | 目的 |
|------|------|
| 法医鉴定 | 确定死亡或致伤原因，认定意外/疾病/自杀 |
| 精神病鉴定 | 确定当事人精神状态、行为能力和责任能力 |
| 刑事科学技术鉴定 | 痕迹鉴定、微量元素鉴定、文书鉴定 |
| 一般技术鉴定 | 工业、农业、建筑、交通运输等专业技术问题 |
| 会计鉴定 | 确定账目、支票、簿册、报表真实性 |

**鉴定人条件**：具备专门知识或技能，本人及其近亲属与案件无利害关系。

**不能充当鉴定人的情形**：本案当事人或当事人近亲属、与本案有利害关系、担任过本案证人/辩护人/代理人、与本案当事人有其他关系可能影响公正鉴定。

##### （4）走访

走访是案件调查中最基本、最重要的措施，每案必用。

**走访方式**：可以公开身份出现，也可以隐蔽身份出现（视案件具体情况而定）。

**走访查明内容**：事故发生前后情况、事故真假及性质、事故事实情节、与事故有关的人事物关系、知情人的看法和议论。

**目的**：认定保险事故的真实性。

##### （5）现场模拟实验

适用于疑难案件通过查勘难以定性的情况。

**要求**：实验条件尽量和事故发生当时保持一致（时间、气候、工具、物品），实验可以多次进行。

**禁止**：足以造成危险、侮辱人格、有伤风化的行为。

## 三、具体案件类型的调查方法与案例

### （一）高坠事故

**伤病特点**：体表轻、内部重——多发性骨折和内脏破裂，皮肤仅有擦伤性质损伤。尸体检验可见结膜下、球结膜有出血斑点。

**调查目的**：区分自杀或意外。

**调查项目**：事故查勘

**调查概述要点**：
- 事故发生时间、落地地点（偏僻或闹市）
- 现场明亮程度、有无危险、地面光滑程度
- 楼梯是否狭窄、道路有无障碍物
- 坠落时是否有人听到喊叫
- 坠落高度、角度及与坠落处的距离
- 有无证据显示出险人努力防止坠落
- 坠落后是否清醒、说了什么
- 有无扯破手指甲或擦伤手
- 坠落时衣着情况（整齐或凌乱、是否穿鞋）
- 出险人出现在现场的原因及活动情况
- 口中有无异物
- 出险人之前是否到过该处并熟悉该处

**案例（高坠事故）**：

李某，女性，29岁，从宾馆三楼坠下，先称意外失足，后称被男服务员推下。

- **现场查勘**：窗台高1.30M，窗前有小桌，屋内物品整齐无搏斗痕迹（反常）；窗台无足迹，外墙窗沿下1.6M和1.65M处有两道新鲜蹬踏划痕；楼下距落地点2M处有女士皮鞋一双和皮包一个
- **医院查勘**：诊断双下肢骨折，双手腕部有对称划伤（反常——一般坠楼不会形成此伤）
- **走访了解**：李某当日和一男子进入房间，二人非夫妻关系。后续询问得知：约会时被男服务员发现敲门，担心被曝光自己跳楼，因害怕无人承担责任所以陷害男服务员

**结论**：故意行为，非意外或他人所为，保险公司不承担保险责任。

### （二）交通事故

**损伤类型**：冲击伤（保险杠损伤）、碾压伤（轮带印痕、伸展伤、皮下剥离创伤）、二次损伤、挥鞭样损伤。

**调查项目**：事故查勘、基础查勘

**调查概述要点**：
- 车辆速度及行驶方向、碰撞部位
- 碰撞后车辆位置和身体位置
- 有无滑行痕迹（长度及位置）
- 有无爆破轮胎或其他破损
- 是否系安全带
- 驾驶者及乘客姓名
- 目击证人及其描述
- 路况（巷道、岔道、弯路、交叉路口、坡道）
- 路面干湿程度、有无障碍物、天气、能见度
- 车辆型号、行驶证有效状况、所有人
- 刹车是否正常、窗户是否清洁
- 出险人既往健康状况（晕眩、心脑血管疾病等）
- 是否熟悉事发地点
- 意外发生时是否因工外出、目的、职业/工作性质

**查勘重点**：
- 现场查勘为主，走访交警部门和处理人员以及救治医疗机构
- 了解详细专业的事故认定结论
- 怀疑酒后驾驶的，事前必须要求家属配合或同意进行血液酒精含量测定
- 基础查勘核实是否存在自杀或故意行为

### （三）溺水事故

**尸体征象**：口鼻部泡沫，男性多俯卧位，女性多仰卧位，手指间多抓有水中植物。

**性质判断**：多为自杀或意外（含他杀），应结合案情和现场调查综合分析。

**调查项目**：事故查勘

**调查概述要点**：
- 确认法医鉴定结论
- 有无目击证人、证人描述
- 当时有无呼救
- 是否会游泳及技术如何
- 出现在出险水域原因
- 是否熟悉该水域情况
- 尸体距离岸边多远
- 岸边有无出险人遗留物
- 衣着情况、有无外伤
- 水域情况（水深、水温、水流速度、有无暗流、水草、旋涡、暗礁）
- 有无救援出险人
- 船只情况（型样、大小、登记号码、救生设备、管理人等）
- 机械设备是否正常运行、溺水时船速、甲板是否光滑、有无护栏

**案例（溺水事故）**：

某大学外语系女学生，在火车站护栏外水塘中发现尸体。

- **现场查勘**：随身物品中无车票。落水点护栏高1.8M，外为40公分宽土沿，再外为陡峭斜坡，下为深2M水塘。土沿潮湿长青苔较滑
- **走访了解**：当地许多人坐火车不买票，翻护栏走土沿出站。事发当天21:35到站，天黑阴天无月光，能见度极低，前两天刚下大雨积水深达2M
- **尸检**：全身无钝器或锐器损伤，双手无搏斗挣扎痕迹
- **家属询问**：从学校几乎一月回家一次，学生证半价票只容许一年四次，无票上车走护栏出站可能性极大，平时不会游泳

**结论**：夜黑能见度低，雨后斜坡较滑，意外失足落水溺死，属于意外身故保险责任。

### （四）不如实告知

不如实告知是保险法和保险合同约定的主要内容，是否存在未如实告知对赔案审核有至关重要的意义。

**调查项目**：正面查勘、医疗查勘、基础查勘

**调查概述**：

1. **正面查勘**：
   - 电话或会面业务员并取得书面声明
   - 确定其招揽业务方式、与出险人关系、认识多久
   - 再次确认业务员对出险人健康状况的描述
   - 确定出险人死因及是否有重要健康问题
   - 会面受益人

2. **医疗查勘**：
   - 会见出险时治疗的主治医生
   - 会见曾经治疗过的主治医生
   - 取得所有诊疗病史记录（现在或过去的疾病）包括检查检验报告
   - 详细查勘出险人住所附近的医院
   - 累积形成完整的出险人医疗病史

3. **基础查勘**：
   - 会见单位或雇主，确定工作习性、缺席及健康记录、职业危险
   - 检查有无其他健康保险、团体保险保障
   - 确认吸烟记录（若与费率有关）
   - 检查申请书财务信息，寻求企业/个人收入的缴税记录、工资证明、破产记录与民事诉讼
   - 确认出险时职业与投保时职业是否有较大差别

**案例（不如实告知——胆囊癌）**：

2003年11月，徐某为其母亲陈某投保美满人生保险10万元，体检后标准承保。2005年2月报案称陈某在家中病故。

- **正面查勘**：家中正在办丧事，家人和村民证实死亡属实，县殡仪馆辨认死者确为投保体检的陈某本人。业务员和受益人称陈某投保前有胆囊手术，无其他健康问题
- **医疗查勘**：在当地医院走访排查，最终在县医院查到陈某在投保前5个月住院治疗，经病理检查确诊"胆囊癌"
- **基础查勘**：陈某是当地偏僻农村老人，平时就医较少，无特殊异常

**结论**：投保人徐某主观上知道陈某已确诊胆囊癌而未告知，客观上投保时填写健康询问时有未告知事实存在。故意未如实告知，保险人解除合同且不退还保费。

### （五）意外伤害

保险事故的原因（意外或疾病）是审核赔案结论的重要依据。

**调查项目**：事故查勘、基础查勘

**调查概述——事故查勘**：
- 事故发生的时间、地点、原因
- 周围环境状况、有无除外责任
- 现场有无目击者及所见所闻
- 走访现场附近住家
- 事故对出险人造成的损害、伤势面积大小
- 事故发生时家属（特别是受益人）在何处
- 处理事故的部门、经办人、出具过的证明材料
- 是否进行过救治、就诊医院
- 住家附近医院、各大医院排查
- 参加救治和处理事故的人员及了解的情况
- 事故双方或关系人查证

**调查概述——基础查勘**：
- 出险人身份是否确认为被保险人
- 是否熟悉事发地点
- 意外发生时到达现场的原因、出事前行踪
- 有无疾病引起可能
- 年龄和一般身体状况，有无眩晕、昏厥、疾病发作、视力或听力损伤等
- 是否饮酒或服用药物
- 事故前言行是否正常
- 家庭关系、经济状况
- 出险人当时职业或工作性质

**案例（意外伤害——床上落地死亡）**：

2005年8月，陈某为其儿子魏某投保福如东海5万元。2005年11月28日报案称魏某在家中从床上摔下，送医抢救无效死亡，申请意外死亡保险金5万元。

- **事故查勘**：死亡属实，已土葬。母亲从厨房回到房间发现魏某自床上落地已神志不清，立即送市医院（反常：市医院较远，县医院较近）。医师在门诊诊断死亡，返家途中将小孩尸体抛入河中（无法尸检）。无其他目击者，门诊病历无外伤记录和诊断
- **基础查勘**：走访业务员、魏某父母及当地医疗机构发现——业务员从未面见过魏某，对其健康状况不了解；投保人家中经济一般，魏某是独子；当地市医院记录显示投保前2月已明确诊断"先心"（先天性心脏病），投保后2月继续在该医院住院治疗且病情危重

**结论**：典型疾病死亡而谎称意外死亡，同时存在故意未如实告知行为。

### （六）重大疾病

重大疾病是重疾保险的主要保险责任，认定是否属于保险责任或责任免除，以及是否存在逆选择是理赔查勘的重点。

**调查项目**：医疗查勘、基础查勘

**调查概述——医疗查勘**：
- 访问诊治医师、查阅病历记录、核对病历笔迹
- 询问出险人的诊治情况、病史、健康状况、发病过程
- 查看病情检验有关记录（血液化验单、X光片、心电图、B超等）
- 核对医疗费用单据
- 了解诊治医师、化验或检验人员与出险人的关系
- 医院规模、设备、除外责任就诊医院的查证

**调查概述——基础查勘**：
- 审核投保书，出险人对询问事项回答情况、是否被保险人签名
- 询问业务员对出险人情况的了解及工作失误
- 对大额保险或有疑问的，普查重点是出险人所在附近医院投保前一两个月前后的检查记录
- 有无任何住院、精神病院、疗养院、理疗计划
- 是否曾因医疗原因致工作中断
- 收集投保时检查记录和出险时的检查、用药记录
- 疑难问题可聘请有关专家"会诊"
- 出险人家属病史、有无酒精或药物滥用证据
- 出险人饮食、运动嗜好
- 查明出险人患病期间的生活、工作情况，行踪与所患疾病有无矛盾
- 附近群众对该事件的议论

**案例（重大疾病——宫颈癌）**：

2004年9月，王某为自己投保重大疾病保险5万元。2005年10月报案称诊断"宫颈癌"住院手术治疗。

- **医疗查勘**：出险经过为单位女工体检宫颈刮片检查。提交的病理诊断报告原件为"宫颈原位癌"。调阅住院病历发现术后全标本病理检查描述为"…粘膜下层可见少量癌细胞浸润…"
- **基础查勘**：该单位约2年进行一次女工体检，上次检查结果良好无就医建议。投保书本人签字认可，了解健康告知内容

**知识要点**：原位癌是上皮恶性肿瘤局限在皮肤或粘膜内，未通过基底膜侵犯到周围组织，是癌的最早期（0期癌），手术切除即可完全治愈。重疾保险将其列为免除责任。常见于子宫颈、皮肤、支气管、胃、前列腺和乳腺等部位。

**结论**：王某此次出险应明确为"宫颈癌"。提交的"原位癌"诊断报告属门诊检取少量标本所致，确诊结论应以手术后全标本为最后结论。不予认定为原位癌免赔。
