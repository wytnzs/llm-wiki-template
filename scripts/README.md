# scripts 目录说明

本目录包含历史迁移、判例分类、判例视图生成等辅助脚本。

重要提醒：

- 其中部分脚本仍引用旧路径，例如 `04判例库/`、`04判例库-按疾病分类/`。
- 当前判例库主路径为 `02-Areas/判例库/`。
- 未核对路径、输入源和输出目录前，不要直接运行这些脚本。
- 公开到 GitHub 时，只保留通用模板脚本或经过脱敏、改成相对路径的脚本。
- 大批量移动、删除、硬链接、软链接类脚本，必须先在测试目录运行。

建议后续整理方式：

1. 把仍有价值的脚本改造成相对路径版本。
2. 把只用于一次性迁移的脚本移动到 `scripts/_archive/`。
3. 为每个可运行脚本补充用途、输入、输出、风险说明。
4. 不把包含真实本地路径或业务数据的脚本上传到公开仓库。
