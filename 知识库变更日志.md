## [2026-07-17] architecture | 简化知识库运营结构

- README 调整为统一使用入口，明确主题、聚合、主题页、关系导航和知识库地图的分工；

- 新增 `00-Inbox/AI待确认.md`，承接不能自动写入正式知识的建议；
- 6 张正式主题页的核心问题改为具体、可回答的问题；
- 判例事实、合同条款、病历及法律/医疗关键结论继续实行人工确认边界。
