# 风格选择器

> 排版和配图前必读。先判断内容气质，再选视觉方案。
> 核心原则：**内容决定形式，不是模板决定内容。**

---

## 一、内容气质判断

写完后，回答三个问题，确定这篇文章的**类型标签**：

### Q1：这篇文章的核心是什么？

| 答案 | 内容类型 |
|------|---------|
| 讲一个真实案例/理赔翻盘 | `案例文` |
| 介绍一个工具/系统/方法 | `工具文` |
| 输出一个观点/判断/认知 | `观点文` |
| 分享个人经历/感悟/复盘 | `故事文` |
| 拆解一个流程/操作步骤 | `教程文` |
| 解读政策/行业动态 | `解读文` |

### Q2：这篇文章的情绪底色是什么？

| 感受 | 情绪标签 |
|------|---------|
| 理性、专业、让人放心 | `冷静` |
| 反常识、让人"原来如此" | `冲击` |
| 温暖、共鸣、像聊天 | `亲和` |
| 简洁、不啰嗦、说清楚 | `利落` |

### Q3：这篇文章想让读者做什么？

| 目的 | 行为标签 |
|------|---------|
| 让他知道一个方法 | `认知改变` |
| 让他信任你这个人 | `信任建立` |
| 让他尝试使用工具 | `行动引导` |
| 让他转发/分享 | `传播驱动` |

### 最终标签示例

```
案例文 × 冷静 × 信任建立
观点文 × 冲击 × 认知改变
故事文 × 亲和 × 信任建立
工具文 × 利落 × 行动引导
```

## 二、风格匹配表

根据内容类型的标签组合，选择视觉方案：

### 案例文 / 冷静 / 信任建立

| 维度 | 选择 |
|------|------|
| **配色** | 深海蓝 `#102542` + 白底 `#FFFFFF`，辅以橙色 `#F77F00` 局部强调 |
| **字体** | Noto Sans SC（正文）+ Noto Serif SC（标题/金句） |
| **卡片风格** | MINI 极简 / SWIS 瑞士国际主义 |
| **排版调性** | 传统商务感，干净利落，留白适中 |
| **封面方案** | 深蓝系（默认），左标题右标签，平静有力 |
| **标题风格** | 陈述结论型 / 具体数字型 |
| **典型适用** | 理赔翻盘案例、条款解读、拒赔分析 |
| **不适用** | 深色背景卡、大面积几何装饰 |

### 案例文 / 冲击 / 传播驱动

| 维度 | 选择 |
|------|------|
| **配色** | 深蓝 `#0A1931` + 橙色 `#F77F00` 大色块，高对比 |
| **字体** | Noto Sans SC（粗） |
| **卡片风格** | BOLD 大胆现代 / 暗底对比卡 |
| **排版调性** | 视觉冲击力强，大字突出反常识结论 |
| **封面方案** | 深蓝系，反差大的标题布局 |
| **标题风格** | 颠覆认知型 / 结果前置型 |
| **典型适用** | "法院判了"类翻盘、反常识判例 |
| **不适用** | 淡色渐变、温柔质感 |

### 工具文 / 利落 / 行动引导

| 维度 | 选择 |
|------|------|
| **配色** | 深色科技风 `#1a1a2e → #16213e` + 蓝紫强调 `#537FE7`，或浅灰 `#f7f8fa` + 深色强调 |
| **字体** | Noto Sans SC + Space Mono（序号/数据） |
| **卡片风格** | NEOF 新未来主义 / SWIS 瑞士国际主义（流程图） |
| **排版调性** | 现代感，结构清晰，分层明确 |
| **封面方案** | 深色科技感，英文标签，几何装饰 |
| **标题风格** | 指出问题型 / 清单型 |
| **典型适用** | AI系统拆解、工具教程、方法流程 |
| **不适用** | 传统商务蓝、衬线字体 |

### 观点文 / 冲击 / 认知改变

| 维度 | 选择 |
|------|------|
| **配色** | 暗黑 `#0d0d0d → #1a1a2e` + 亮色强调（青 `#00d4ff` / 紫 `#c442f5`） |
| **字体** | Space Mono（标签）+ Noto Sans SC Bold |
| **卡片风格** | BOLD variant（深色对比）/ 暗底对比卡 |
| **排版调性** | 高冲击力，对撞感，两张暗底卡制造节奏变化 |
| **封面方案** | 深色反差，BEFORE/AFTER 或 SHIFT 主题 |
| **标题风格** | 反常识 / 颠覆认知 / 对比反差 |
| **典型适用** | "最值钱的不是AI"类、行业判断、认知输出 |
| **不适用** | 明亮色、温和留白、传统商务 |

### 故事文 / 亲和 / 信任建立

| 维度 | 选择 |
|------|------|
| **配色** | 米白 `#FAFAF7` + 茶色 `#8b7a6a` + 琥珀 `#c4814a` |
| **字体** | Noto Serif SC（正文/标题）+ Noto Sans SC（说明文字） |
| **卡片风格** | JPNM 日式极简 / WARM 温暖人文 |
| **排版调性** | 留白大、节奏慢、读起来像翻一本笔记 |
| **封面方案** | 暖色系（奶油橙/墨绿），简短标题，有故事感 |
| **标题风格** | 共鸣型 / 代入视角 |
| **典型适用** | 个人经历、创业复盘、成长感悟 |
| **不适用** | 科技蓝、暗黑背景、流程图 |

### 教程文 / 利落 / 行动引导

| 维度 | 选择 |
|------|------|
| **配色** | 浅灰 `#f7f8fa` + 深灰标识 `#1a2236` |
| **字体** | Noto Sans SC + Space Mono 序号 |
| **卡片风格** | SWIS 瑞士国际主义（流程图/步骤） |
| **排版调性** | 信息层级分明，步骤清晰，节奏紧凑 |
| **封面方案** | 简洁商务，数字/步骤突出 |
| **标题风格** | 清单型 / 具体数字型 |
| **典型适用** | 操作指南、步骤拆解、实操教程 |
| **不适用** | 暗黑色系、艺术装饰、大字冲击 |

### 解读文 / 冷静 / 认知改变

| 维度 | 选择 |
|------|------|
| **配色** | 深海蓝商务体系（标准）|
| **字体** | Noto Sans SC |
| **卡片风格** | MINI 极简 / 柱状图（数据对比）|
| **排版调性** | 干净、权威、有数据支撑 |
| **封面方案** | 深蓝标准系，标签含"政策/数据" |
| **标题风格** | 指出问题型 / 具体数字型 |
| **典型适用** | 政策解读、行业数据分析、监管动态 |
| **不适用** | 故事感设计、大面积装饰 |

## 三、卡片风格交替规则

一篇配图多张时，相邻卡片**先按内容分段的气质选择风格**，如果内容气质相近导致相邻风格相同，做局部调整：

| 相邻情况 | 做法 |
|---------|------|
| 同类型内容连续出现 | 用同一风格但交换配色（深色↔浅色） |
| 内容气质有转折 | 风格跟着内容气质切换 |
| 无法判断 | 深色→浅色→暖色→深色，自动循环 |

## 四、使用流程

```
写完全文后：
↓
Step 1：判断内容类型（Q1）、情绪底色（Q2）、行为目的（Q3）
↓
Step 2：输出标签组合（如"观点文 × 冲击 × 认知改变"）
↓
Step 3：查风格匹配表，确定排版调性、配色、字体、卡片风格、封面方案
↓
Step 4：如果多张配图，按交替规则安排风格顺序
↓
Step 5：进入排版+配图执行
```

> **关键检查**：生成完配图后，回看第一张，问自己——读者看到这张配图的第一眼感觉，和文章标题的第一感觉，是一回事吗？如果不是，换风格。
