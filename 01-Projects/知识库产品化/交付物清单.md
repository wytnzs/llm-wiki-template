---

## 核心交付（所有版本包含）

### 1. 目录结构模板

```
00-Inbox/              ← 输入入口
  得到大脑/             得到笔记导入
  灵感随手记/           AI直写
  网页剪藏/             浏览器插件
01-Projects/           ← 当前项目
  你的项目名称/         自定义
  自媒体运营/           内容产出管理
02-Areas/              ← 长期资产
  知识卡片/            原子化知识
  判例库/（可选）       结构化证据
  主题聚合/            跨卡片组织
03-Resources/          ← 外部资料
04-Archive/            ← 归档
06-选题库/             ← 内容入口
   cards/              选题卡片
   选题总索引.md
```

**交付形式**：文件夹结构 + README 说明文件。

### 2. 知识卡片模板（3套）

| 模板 | 适用场景 | 字段数 |
|------|---------|--------|
| concept模板 | 概念/定义类 | 10个字段 |
| method模板 | 方法/流程类 | 12个字段 |
| principle模板 | 原则/判断标准类 | 8个字段 |

**交付形式**：Markdown 模板文件 + 3个已完成的示例卡片。

| skill | 用途 | 包含 |

| gzh-pipeline | 公众号写作全流程 | 认知引擎 → 选题 → 调研 → 写作 → 审校 → 排版配图 → 朋友圈 |

**交付形式**：SKILL.md 模板文件 + 修改指南。

### 6. 选题库五棵树框架

| 选题树 | 定位 | 举例 |
|--------|------|------|
| 工具实操 | 展示专业方法 | "XX工具从入门到精通" |
| 案例拆解 | 用案例证明能力 | "一个真实案例告诉你XX" |
| 方法论 | 分享思维框架 | "我做XX的三个原则" |
| 趋势认知 | 解读行业变化 | "2026年XX趋势" |
| 个人故事 | 建立情感连接 | "我为什么从XX转到XX" |

**交付形式**：选题树说明 + 种子选题清单（每棵树至少3个方向）。
