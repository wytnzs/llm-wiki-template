# 当前迭代计划

## 已完成

- [x] 确定 README 为唯一必需入口；
- [x] 确定目录是角色建议，不是强制分类；
- [x] 确定 `title`、`type` 为最小字段；
- [x] 将知识库地图、判例库和选题库改为可选模块；
- [x] 公开模板增加安装前询问与隐私边界；
- [x] 清理本地旧路径、旧索引、缓存和一次性脚本。

## 下一阶段

- [ ] 用全新空目录完成一次标准安装演示；
- [ ] 用一个已有知识库完成一次“保留原结构”的改造演示；

- [ ] 收集首批用户反馈，优先改 Skill 和安装说明，不急于增加目录。

## 完成标准

用户只提供 GitHub 地址和基本需求，Claude Code 能先询问、再安装；不上传私人数据，不强制选题库，不创建无用索引，并能用 README 解释最终结构。
