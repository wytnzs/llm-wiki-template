# 直播售卖大纲

> 总时长约 35-40 分钟。节奏：痛点 → 方案 → 演示 → 转化。

---

## 第一部分：问题诊断（5分钟）

**目标**：让观众觉得"说的就是我"。

### 开场钩子（30秒）

> "你有没有这种感觉——做了好几年，电脑里的资料越攒越多，但真要用的时候，什么都找不到？"

### 四个灵魂拷问

| 问题 | 戳中谁 |
|------|--------|
| "你的微信收藏夹里存了多少篇从来没看过的文章？" | 所有人 |
| "每次写文章是不是都从零开始？查半天资料，写半天，发出去效果还不好。" | 想内容的人 |
| "你学了很多课、看了很多书，但过两周就想不起来学了什么？" | 持续学习的人 |
| "你有没有一个地方，能让你8年的经验随时调出来用？" | 资深从业者 |

**过渡**："如果这些你都有——别担心，你不是一个人。而且，有一个解决方案。"

---

## 第二部分：解决方案（8分钟）

**目标**：讲清楚"知识资产生产线"是什么。

### 核心比喻

> "大多数人做知识管理，是在做'仓库'——东西放进去就堆着。我做的是'工厂'——原材料进来，加工成资产，然后产出内容。"

### 五层架构（3分钟）

```
输入层 → 资产层 → 项目层 → 输出层 → 反馈层
```

每层一句话：
- **输入层**：有意识收集，不是什么都存
- **资产层**：加工成原子化知识卡片
- **项目层**：围绕目标组装知识
- **输出层**：文章/视频/决策
- **反馈层**：每次输出都让系统变强

### content-brief 交接机制（2分钟）

"这里最关键的设计是 content-brief。它不是笔记，是内容生产的交接单。"

> "写一篇文章之前，先回答五个问题：写给谁？讲什么？凭什么讲？调什么判例？不能说哪句话？回答清楚了，AI帮你写初稿。"

### 三个差异化（2分钟）

1. **不是笔记工具，是内容生产引擎**
2. **有标准流程，不是自由发挥**
3. **深度嵌入AI，不是插件级**

---

## 第三部分：演示闭环（12分钟）

**目标**：让观众看到"从判例到文章"的全过程。

> 详见「演示样例.md」，直播时按此流程走。

### 演示案例

用一条真实的拒赔判例（未如实告知拒赔案）：

| 步骤 | 展示内容 | 时长 |
|------|---------|------|
| 1. 原始素材 | 裁判文书的概览和关键信息提取 | 2分钟 |
| 2. 知识卡片 | 从案例中提炼"未如实告知的判断边界"卡片 | 2分钟 |
| 3. 主题聚合 | 拉出"保险理赔主题地图"显示上下文 | 1分钟 |
| 4. 选题卡 | 生成"未如实告知≠一定赔不了"的选题卡 | 2分钟 |
| 5. content-brief | 写brief，展示字段 | 2分钟 |
| 6. 成品输出 | 公众号文章初稿 | 2分钟 |
| 7. 效果反哺 | 文章发布后的复盘回写 | 1分钟 |

**关键演示技巧**：每一步都要让观众看到"原来可以这么简单"的表情。

## 第四部分：产品介绍与转化（8分钟）

### 产品分层

| 版本 | 适合谁 | 内容 | 价格带 |
|------|--------|------|--------|
| **入门版** | 个人使用者 | 目录结构模板 + 知识卡片模板 + content-brief模板 | 低 |
| **标准版** | 内容创作者 | 入门版 + skill流程模板 + 选题库框架 | 中 |
| **完整版** | 专业团队 | 标准版 + 判例库三视图方案 + 复盘反哺机制 | 高 |

### 限时权益

- 前10名：一次1对1搭建指导（30分钟）
- 全套模板持续更新
- 社群答疑

### 常见反对处理

| 反对 | 回应 |
|------|------|
| "太复杂了，学不会" | "看起来复杂，但核心其实只有三步：记卡片、写brief、产出。我可以带你走一遍。" |
| "我用Notion就够了" | "Notion是笔记本，我这是工厂。如果你只是记笔记，Notion完全够用。但如果你需要持续产出内容——这是一个完全不同的赛道。" |
| "我哪有时间搭这个" | "模板都给你准备好了，你要做的就是往里填内容。而且AI会帮你做大部分工作。" |
| "价格有点贵" | "想想你一年花在课程上的钱。学完就忘。而这个系统，每一分钱都在帮你沉淀可复用的资产。" |

## FAQ预判

| 问题 | 回答要点 |
|------|---------|
| "这个适合我吗？" | 适合需要持续输出内容、沉淀专业知识的个人或小团队 |
| "需要花多少时间搭建？" | 模板已准备好，1-2天搭建骨架，持续填充内容 |
| "和Notion/Roam有什么区别？" | 笔记本 vs 工厂的差异。本系统自带内容生产管线 |
| "我不会用AI怎么办？" | 流程设计为"人决策，AI执行"，不需要写代码 |
| "可以退款吗？" | 提供7天试用，不满意退全款 |
