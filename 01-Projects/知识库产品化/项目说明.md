# 知识库产品化项目

## 定位

把个人知识库从资料仓库升级为**知识资产生产线**。
适用于需要长期积累、输出、决策、教学、咨询、内容生产的人群。

## 核心方法论

`
输入层：Inbox / Resources
资产层：Concept / Relation / Principle / Method / Action
项目层：Projects
输出层：内容 / 产品 / 决策
反馈层：复盘 / 更新 / 反哺
`

## 项目目标

1. 形成可复用的知识管理系统
2. 打磨成可直播售卖的产品
3. 建立知识资产化SOP
