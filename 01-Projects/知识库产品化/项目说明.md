# 知识库产品化项目

## 定位

把个人资料库升级为一套可由人和 Claude Code 共同维护的知识资产系统。产品交付的不是固定文件夹，而是最小标准、初始化方法、可选模块和持续养护 Skill。

## 最小标准

1. 根目录 `README.md` 是唯一必需入口；
2. 输入、项目、资产、资源、归档是通用角色，目录可调整；
3. 正式知识文件至少有 `title`、`type`；
4. 知识库地图、判例库、选题库按用户需求安装；
5. Claude Code 在修改已有知识库前先询问，不上传私人数据。

## 产品目标

- 用户能把仓库地址交给 Claude Code 完成初始化；
- 已有知识库能保留原结构并建立清晰角色映射；
- 日常形成“输入、整理、连接、使用、复盘”的闭环；
- 失败能追溯到知识、项目定义、模板或 Skill 并持续修正。

## 当前边界

当前版本以本地 Markdown、Obsidian 和 Claude Code 为基础，不承诺自动联网、RAG、数据库或从知识卡片自动生成正式选题。
