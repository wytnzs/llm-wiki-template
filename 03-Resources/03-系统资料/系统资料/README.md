# 系统资料导读

本目录保存系统如何演进到当前状态的历史依据，不承担当前规则入口。当前规则以根目录[[README]]、现行 Skill 和项目状态文件为准。

## 内容分工

- `系统设计历史/`：旧规格、旧索引、设计方案和迁移数据，用于解释历史决策与必要时恢复；
- `知识库项目/`：早期知识库建设记录，用于复盘产品方法，不作为现行操作说明；
- [[03-Resources/系统资料/04目录迁移记录-2026-07-20|04目录迁移记录]]：记录本次统一来源层调整和恢复路径。
