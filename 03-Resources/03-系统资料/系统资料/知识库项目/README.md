# 📎 知识库项目

基于 Karpathy 三层架构（Raw→Wiki→Query）+ Zettelkasten 卡片笔记法 + GBrain 长期记忆思想 + OKF-like 元数据规范搭建的 AI 知识资产生产系统。

## 结构

- `00-Inbox/` — 信息入口，只存不整理
- `01-Projects/` — 进行中的项目
- `02-Areas/` — 长期知识资产：知识卡片、判例库、主题聚合
- `03-Resources/` — 外来参考材料
- `04-Archive/` — 归档
- `05-Skills/` — 流程模板
- `06-选题库/` — 选题资产

## 项目文档

- [[三套知识库思想整合规划]] — LLM Wiki、GBrain、Gemini OKF 三套思想的整合规划和下一步优化计划

原子化 | 双向链接 | 主题长期记忆 | 资产溯源 | 输出反哺
