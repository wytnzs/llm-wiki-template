# 个人输入来源导读

本目录保存用户本人输入和得到大脑笔记。这里的内容是来源，不因已被提炼而失效；原始表达保持不变，通过具体来源路径连接正式知识资产。

## 已确认的转化关系

- [[03-Resources/个人输入/得到大脑已处理/注意力在用户身上-素材|注意力在用户身上素材]] → [[02-Areas/知识卡片/认知/注意力在用户身上增长才有可能|注意力在用户身上，增长才有可能]]：原始素材被提炼为用户研究与内容增长判断。
- [[03-Resources/个人输入/得到大脑已处理/经验资产化-素材|经验资产化素材]] → [[02-Areas/知识卡片/认知/经验资产化-从完成工作到沉淀系统|经验资产化]]：原始工作方式判断被扩展为技能、护栏和记忆结构。
- [[03-Resources/个人输入/得到大脑已处理/客户买的是确定性-素材|客户买的是确定性素材]] → [[02-Areas/知识卡片/商业/客户买的是确定性|客户买的是确定性]]：保留原始表达，同时以正式卡片维护结果保证边界。
- [[03-Resources/个人输入/得到大脑已处理/好的学习是有摩擦的  突破舒适区学习|好的学习是有摩擦的素材]] → [[02-Areas/知识卡片/认知/好的学习是有摩擦的|好的学习是有摩擦的]]：原始短句由正式卡片补充适用边界。

## 使用边界

- `archive_note`、`归档`等旧元数据保留为历史导入痕迹，不代表当前目录角色；当前角色由本导读统一定义为“个人输入来源”；
- 不把每条输入都机械制成知识卡片；只有能跨项目帮助判断或行动的内容才进入资产层；
- 个人判断与外部事实分开，引用外部数字、案例和专业结论时仍需回到原始出处复核。
