# 营销革命4.0-营销应用提炼

## 核心方法：5A顾客路径

1. **了解（Aware）**：受众第一次知道你和这门课。
2. **吸引（Appeal）**：某个场景、冲突或结果让他觉得“这和我有关”。
3. **问询（Ask）**：他主动搜索、评论、私信或向同行求证。
4. **行动（Act）**：报名、上课并实际使用工具。
5. **拥护（Advocate）**：因为真实获得结果，愿意复购或推荐给同行。

数字环境中的营销不是一次成交，而是受众在个人判断、他人意见和真实体验之间不断移动的过程。传播内容、私聊回复、课程体验和课后反馈属于同一条顾客路径。

## 对保险实务课程营销的应用

| 阶段 | 课程动作 | 观察指标 |
|---|---|---|
| 了解 | 短视频、朋友圈判断题 | 精准受众是否停留或回应 |
| 吸引 | “收到通知书后下一步怎么做”的场景承诺 | 收藏、转发、关键词回复 |
| 问询 | 免费自查卡、15分钟公开拆解 | 有效私聊与具体问题数量 |
| 行动 | 报名、课堂完成判断表 | 到课率、作业完成率 |
| 拥护 | 7天作业、反馈、同行转介绍 | 独立使用率、真实推荐 |

## 边界

5A不是固定的线性漏斗；受众可能跳步或返回。不能只优化曝光和报名而忽略课程使用结果。
