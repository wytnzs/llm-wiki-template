# 告知的艺术与理赔的技巧（六）—— 未如实告知拒赔的热点问题

> **日期**：2026年3月12日 | **讲师**：刘立群

---

本课讲解四个与告知相关的热点问题：两年不可抗辩条款、概括性问询、医保卡外借、多家投保与拒保延期问询。

## 一、两年不可抗辩条款深度解析

### 历史与发展

| 时间 | 事件 |
|------|------|
| 19世纪初 | 英国伦敦，商人老罗德案——投保时签署保证声明，10年后因肝病去世，保险公司以"年轻时患过肺结核"为由拒赔 |
| 1848年 | 英国伦敦信用寿险公司首次承诺：保单生效超过两年，放弃任何情况下保单抗辩的权利 |
| 1864年 | 美国曼哈顿人寿将不可抗辩条款作为合同固定条款 |
| 1906年 | 美国纽约州《阿姆斯特朗法案》将其上升为强制法律规范 |
| 1946年 | 美国出台《不可抗辩示范法》，要求全美所有州的人寿保单必须含有不可抗辩条款 |
| 1995年 | 中国颁布《保险法》，仅规定了如实告知义务和解除权，尚未写入不可抗辩条款 |
| 2009年 | 中国修订《保险法》，正式写入两年不可抗辩条款 |

### 设计的初衷

两年不可抗辩条款源自**保险的最大诚信原则**，目的：
- 维持保险的稳定性和公信力
- 平衡保险公司和客户双方的权益
- 既赋予保险公司一定时间内对不告知行为的抗辩权
- 又保护客户的长远利益，限制保险公司任意解除合同和拒赔

### 国际对比

| 国家/地区 | 适用范围 | 限制条件 |
|-----------|----------|----------|
| **美国** | 寿险、健康险、意外险 | ①保险欺诈不适用 ②须被保险人在保单生效两年内生存且未发生约定事故 |
| **中国** | 全部险种 | 法律条文较简略，实际判例有差异 |
| **香港** | 普遍适用于寿险；意外险/重疾险/医疗险普遍不适用 | 同英美法系 |

> **海港城事件**：2015年大陆客户在香港为孩子投保重疾险，未告知2014年孩子因呼吸道感染住院的病史。2017年保单生效两年后确诊白血病，香港保险公司拒赔（因香港重疾险普遍不适用不可抗辩条款）。若发生在中国大陆，极可能援引两年不可抗辩条款获得赔付。

### 两年不可抗辩条款不适用的情况

#### 1. 恶意欺诈

两年不可抗辩条款不能成为**恶意欺诈骗保者的法律保护伞**。司法对欺诈行为态度明确：不适用。

#### 2. 保险责任条款不适用

重疾险条款通常约定："被保险人在等待期后，由本公司认可的医生确诊**首次发生**本合同所定义的重大疾病。"

**典型场景**：客户投保前已确诊乳腺癌并手术+化疗，投保时未告知。保单生效两年后乳腺癌复发 → 保险公司可以"不符合保险责任条款（非首次发生）"拒赔，两年不可抗辩条款管不到。

#### 3. 免责条款不适用

**先天性疾病**在医疗险和重疾险中属于免责条款，无论保单生效多少年，免责条款始终有效，不受两年不可抗辩影响。

#### 4. 保险事故发生在两年内，两年后申请理赔

**最高院公报案例（2015年12月）**：

| 项目 | 内容 |
|------|------|
| 投保 | 2010年8月25日（出院第二天投保，恶意欺诈） |
| 出险 | 2012年6月起因肺癌先后9次住院（两年内） |
| 申请理赔 | 2012年9月11日（保单生效超两年后） |

**法院裁判**："自合同成立之日起超过两年保险人不得解除合同"的前提是——合同成立之日起**两年后新发生的**保险事故。

#### 相关案例：投保前已患病

秦某2010年9月确诊左股骨肉瘤并手术+多次化疗，2015年4-6月复发化疗。2015年10月18日投保重疾险（未告知）。保单生效两年后因"左股骨肿瘤切除术后复发"住院理赔。

**法院判决**：被保险人在保险合同成立以前已经患病，不属于保险责任范围，不适用于不可抗辩条款。支持保险公司。

### 对代理人的建议

1. **两年不可抗辩条款应在理赔时用，投保时尽量不用**。它是"盾"（法定的抗辩理由），不是"枪"（投机的工具），更不是骗保的工具。
2. **如果要利用此条款，需充分考虑风险**：
   - 保险公司加强了两年内的保单筛查
   - 两年内出险的调查力度非常大
   - 还有上述四种不适用的情况
3. 风险何时发生是不确定的（这正是买保险的原因），不要假设客户两年内不会出险。

## 二、概括性问询的司法实践

### 两个高度相似、结果相反的案例

#### 案情对比

| | 上海案 | 青海案 |
|------|--------|--------|
| 投保时间 | 2018年4月 | 2017年6月 |
| 问询 | "过去三年内是否曾有医学检查结果异常" | "过去三年内是否有医学检查包括健康检查结果异常" |
| 客户回答 | 否 | 否 |
| 实际有 | 2016-2017年门诊查出甲状腺结节 | 2017年6月19日体检发现甲状腺结节 |
| 出险 | 2018年12月确诊甲状腺癌 | 2017年11月确诊甲状腺癌 |
| 客户主张 | 该问询属概括性问询，不用回答 | 该问询属概括性问询、笼统告知 |

#### 判决结果对比

| 上海案 | 青海案 |
|--------|--------|
| "问询内容明确，客户有甲状腺结节属异常结果，应告知" | "为概括性条款，属笼统告知，不够具体明确" |
| 客户**败诉**，支持保险公司 | 客户**胜诉**，保险公司赔偿 |

### 启示

1. **概括性问询没有统一的客观标准**，不同地区、不同法官可能有不同认定
2. 只有"除上述之外的其他疾病"等有明确"其他"字样且无具体限定的问题，才是相对明确的概括性问询
3. 对其他有争议的问询，保险公司一般不接受"概括性问询"的质疑，最终只能到法院
4. **在判断时务必要慎重**，不要仅凭个人感觉下结论

## 三、医保卡外借问题

### 从理赔角度看医保卡

医保卡是有身份印记的卡片（身份证号+医保属性），记录了一个人的就诊信息。理赔时，保险公司以医保卡为调查线索，重点关注**投保前的就诊记录**，而不会关注"医保卡是不是外借的"。

一旦查出投保前存在就诊/检查/诊断/住院记录，理赔就会认为存在投保前异常。

### 不同使用场景的风险等级

| 场景 | 风险等级 | 分析 |
|------|----------|------|
| **药店买药** | 🔵 低 | 几乎没有问询问"是否买药"；药品用途不确定，无法与疾病建立必然联系 |
| 药店频繁买同类药（持续30天以上） | 🟡 中低 | 部分问卷有"持续用药30天"问题，需注意 |
| **门诊就诊** | 🟠 中高 | 会留下检查记录和诊断；客户可能都不知道医生下了什么诊断 |
| **住院** | 🔴 高 | 不仅涉及理赔纠纷，还涉及冒用医保身份、骗取医保统筹基金的违法行为 |

### 自证清白

- ❌ 让借用人做证明 → **不管用**（利害关系人）
- ✅ 提供客观证据：
  - 客户本人的体检报告（投保前后两三年）
  - 直观证据：如医保卡记录是子宫切除，客户去医院做B超证明子宫还在

### 注意当地医保政策

部分地区已实行家庭医保账户共用，就诊记录不混充。但各地方政策不同，要具体分析。

### 成功自证清白的判例

客户2014年3月投保重疾险30万。2015年7月因心脏病行冠状动脉搭桥手术。保险公司查出2010-2012年医保卡多次开心脏病药，诊断有高血压、高血脂等。客户主张"是给父亲开药用的"。

**客户提交的证据**：
- 父亲2009年因心脏病住院的病历
- 2013年12月（投保前）和2014年12月（投保后）两份自身体检报告，证明没有心脏病、高血压
- 2018年电子胃镜报告、肺功能报告

**法院采信**：证据证明医保卡上的记录不属于客户本人 → 不需告知 → 判决保险公司承担保险责任。

## 四、多家投保与拒保延期问询

### 理赔视角

理赔通常**不会首先使用**这一类问题处理案件，而是**先找身体异常**。只有在：
1. 没有找到其他健康异常
2. 且发现客户存在保险欺诈嫌疑

这种情况下，才会用"多家投保未告知"作为拒赔依据。

### 典型欺诈案件特征

- 多家投保
- **短期出险**（等待期刚过就出险）
- **保额超出普通人**
- 普通老百姓一看就感觉"有骗保嫌疑"

### 警示案例（2020-2021年广泛报道）

| 项目 | 内容 |
|------|------|
| 投保 | 2018年8月-2019年1月，在多家公司密集投保重疾险 |
| 累计保额 | **600多万** |
| 关键特征 | 每家都在免体检额度之内；其中一家要求体检时撤件 |
| 出险 | 2019年4月发现甲状腺异常，5月确诊甲状腺癌 |
| 客户背景 | **医学专业**，且在保险公司做过**理赔**和**销售** |
| 抗辩理由 | 未查到投保前具体异常的就诊记录 |

**法院判决**：客户具有医学专业背景和保险销售经验，在各公司免体检额度内密集投保，短期出险，面对多家投保问询时不如实回答 → **属于故意不告知** → 支持保险公司拒赔。

### 对代理人的建议

- 如果客户特别关注"多家投保/延期拒保"问题，或表露不想告知的意图
- 或客户的**收入与保额严重不符**
- → **要引起注意**，引导客户如实告知
- 如果仅是正常寻找宽松核保产品，保额也不特别高，则不告知的风险不大
