# 告知的艺术与理赔的技巧（四）—— 理赔对未如实告知的处理（保险公司视角）

> **日期**：2026年3月12日 | **讲师**：刘立群（前保险公司理赔经理、保险理赔事务所创始人）

---

从本课开始，从**理赔角度**讲解未如实告知。即使我们尽职尽责，实际操作中仍可能出现未如实告知的情况。本课站在**保险公司视角**，了解理赔时如何分析和处理。

---

## 一、法律依据：《保险法》第16条

> **第16条** 订立保险合同，保险人就保险标的或者被保险人的有关情况提出询问的，投保人应当如实告知。
>
> 投保人故意或者因重大过失未履行前款规定的如实告知义务，足以影响保险人决定是否同意承保或者提高保险费率的，保险人有权解除合同。
>
> 前款规定的合同解除权，自保险人知道有解除事由之日起，超过三十日不行使而消灭。自合同成立之日起超过二年的，保险人不得解除合同；发生保险事故的，保险人应当承担赔偿或者给付保险金的责任。
>
> 投保人故意不履行如实告知义务的，保险人对于合同解除前发生的保险事故，不承担赔偿或者给付保险金的责任，并不退还保险费。
>
> 投保人因重大过失未履行如实告知义务，对保险事故的发生有严重影响的，保险人对于合同解除前发生的保险事故，不承担赔偿或者给付保险金的责任，但应当退还保险费。
>
> 保险人在合同订立时已经知道投保人未如实告知的情况的，保险人不得解除合同；发生保险事故的，保险人应当承担赔偿或者给付保险金的责任。

## 二、未如实告知案件的法律处理逻辑

```
未如实告知情况
    │
    ├─ 不影响承保 → 维持合同有效，承担保险责任
    │
    └─ 影响承保
         │
         ├─ 故意未如实告知 → 解除合同 + 拒赔 + 不退保费
         │
         └─ 重大过失未如实告知
              │
              ├─ 与出险事故有关联性 → 解除合同 + 拒赔 + 退还保费
              │
              └─ 与出险事故无关联性 → 可解除合同 + 但须赔付
```

---

## 三、理赔处理时的五大考量要素

### 要素一：有没有问？

**核心判断**："客户存在投保前异常情况，并不意味着发生了未如实告知。"

有限告知原则下，有问才有答。理赔人员首先要拿出投保问卷，确认客户的异常情况是否属于问询范围。

**案例**：客户有高血压、高血脂、糖尿病，代理人帮他投保了**恶性肿瘤疾病保险**（绕开了相关问询）。出险后保险公司以高血压、高血脂、糖尿病未告知为由拒赔。

→ **应对**：先看问卷有没有问。如果问卷没有问这些疾病，投保操作就没问题，可直接挑战保险公司的拒赔决定。

### 要素二：是否影响承保？（转二核）

理赔阶段发现未告知，理赔人员将情况转给**核保人员**进行风险评估（保险公司内部称"转二核"）。

核保结论：

| 结论 | 是否影响承保 | 后果 |
|------|-------------|------|
| 标准体承保 | 不影响 | 保险公司不能解除合同 |
| 加费/除外/延期/拒保 | 影响 | 保险公司有权解除合同 |

**重要澄清**：核保结论（如"除外承保"）仅供理赔人员**参考**，最终决定权在**理赔部门**。按照法律规定，只要影响承保，理赔就可以解除合同。因此：
- 核保说"除外"，理赔直接解除 → 法律上没错
- 核保说"除外"，理赔也做"除外" → 也可以，更通融

### 要素三：主观性质——故意还是重大过失？

#### 什么是故意？
客户**看到了问题，知道要回答，知道回答后会影响承保**，但就是没有回答。即：**有意为之**。

#### 什么是重大过失？
客户看到了问题，但**因疏忽大意，或误认为不用告知/告知了也不影响承保**，而没有告知。即：**疏忽或误解下的未告知**。

#### 如何判断？

理赔人员会综合以下因素：
- 投保过程（自己投保还是代理人协助？）
- 异常情况距离投保的时间（时间短→可疑，时间长→可能是遗忘）
- 异常情况的严重程度和持续时间
- 未告知异常与出险之间的关联性
- 客户的文化程度、工作背景

**保险公司倾向**：大部分公司站在对自己有利的角度，倾向于界定为"故意"——因为处理更简单（直接解除+拒赔+不退费），比"重大过失"路径省去关联性判断和举证责任。

### 要素四：关联性（仅重大过失场景）

如果认定为重大过失未告知，还需判断**未告知事项与出险事故之间是否具有关联性**：

- **有关联性** → 可解约拒赔，但退还保费
- **无关联性** → 可解约，但须赔付

### 要素五：两年不可抗辩条款

保单生效**超过两年**，保险公司不能再追究未如实告知的责任。

**实践影响**：
- 两年内出险 → 调查力度大、审查严
- 超过两年后出险 → 调查率大幅降低

此外，保险公司还会考虑：客户诚信情况、行业环境、监管政策、司法判例等外部因素。

## 四、案例分析：庞某乙肝住院理赔案

### 基本案情

| 项目 | 内容 |
|------|------|
| 投保时间 | 2017年3月8日生效 |
| 险种 | 重疾险20万 + 住院医疗险1万 |
| 出险时间 | 2017年9月27日（生效仅6个月） |
| 出险原因 | 乙型病毒性肝炎住院25天，医疗费18000+元 |
| 调查发现 | 2015年11月曾因腹股沟斜疝住院，期间检出：大三阳、肝功能异常（转氨酶高）、高血压 |
| 客户未告知 | 住院史、大三阳、肝功能异常、高血压 |

### 处理思路推演

1. **有没有问？** → 重疾险+医疗险问卷通常有"五年内是否住过院""三年内是否有体检异常"→ 客户应回答但未回答 → **存在未如实告知**

2. **是否影响承保？** → 腹股沟斜疝已手术，不影响。但大三阳+肝功能异常+高血压 → 严格的公司直接拒保，宽松的可能复查后加费 → **影响承保**，保险公司有权解除合同

3. **主观性质判断？**
   - 住院距离投保间隔一年半，期间无就诊/服药记录
   - 住院病历仅写了"腹股沟斜疝"，未写肝炎和高血压诊断
   - 住院仅5天就出院
   - 综合考虑，**故意不告知的意图不明显** → 认定为**重大过失**

4. **关联性？** → 未告知"大三阳"与本次出险"乙型肝炎住院"**有关联性** → 可解约拒赔，退还保费

5. **对其他险种的影响？** → 保险公司发现未告知，需对该客户名下**所有保单**进行风险评估 → 重疾险也一并解约退费

## 五、保险公司内部处理流程

### 常规流程

1. **理赔发现未告知** → 转二核做风险评估
2. **高级别理赔人员审批**（需要经验+法律背景+纠纷处理能力）
3. **初步决定后→与客户沟通**（发正式通知前）
   - 客户可提出对自己有利的辩解理由
   - 如需要可发起二次调查
   - 可能维持原决定，也可能改变

### 法律程序性规则（必须遵守）

#### 规则一：30日解除权

保险公司知道解除事由后，**30日内**必须行使合同解除权，否则权利消灭。

**解除方式**：必须**通知客户**解除合同。仅口头告知不够。

#### 规则二：拒赔通知书

《保险法》第24条：拒赔决定作出后**3日内**向被保险人或受益人发送拒赔通知书。虽无明确"超时必赔"的后果规定，但超期可作为抗辩依据。

#### 规则三：解除合同与拒赔的关系

拒赔通知书必须**以解除合同为前提**。

**常见低级失误**：拒赔通知书只写了"因未如实告知，本公司决定拒赔"，**没有写解除合同**。一旦超过30天，意味着保险公司未在法定期限内解除合同 → 客户可以此为由要求继续承担保险责任。

## 小结

| 考量要素 | 核心问题 |
|----------|----------|
| ① 有没有问 | 客户的异常是否在问卷范围内？ |
| ② 是否影响承保 | 转二核进行风险评估 |
| ③ 故意/重大过失 | 投保过程、时间间隔、异常严重程度等综合判断 |
| ④ 关联性 | 未告知事项与出险事故之间是否有因果关系？ |
| ⑤ 两年不可抗辩 | 保单是否生效超过两年？ |

**核心法律依据**：《保险法》第16条（建议背诵）。

**关键认知**：保险公司的程序性失误（如30日未解除合同、拒赔通知书写"拒赔"但未写"解除合同"）可成为帮客户争取理赔的有力切入点。
