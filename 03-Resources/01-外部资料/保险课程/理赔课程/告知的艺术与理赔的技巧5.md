# 告知的艺术与理赔的技巧（五）—— 反驳保险公司拒赔的四大理由（客户视角）

> **日期**：2026年3月12日 | **讲师**：刘立群

---

本课站在**客户角度**，梳理反驳保险公司以未如实告知为由拒赔的法律依据和策略。总结归纳为**四大类反驳理由**。

---

## 一、反驳理由一：问询瑕疵

### 情形1：没有相关的问题

**法律依据**：
- 《保险法》第16条第一款：保险人就有关情况**提出询问**的，投保人应当如实告知
- 《保险法司法解释二》第6条：投保人的告知义务**限于保险人询问的范围和内容**。当事人对询问范围和内容有争议的，保险人负有举证责任

**案例**：王某有多年高血压和糖尿病史，投保了**女性特定恶性肿瘤保险**（仅保障乳房、子宫、卵巢、阴道四个部位的恶性肿瘤）。保单生效一年后确诊乳腺癌，保险公司以未告知高血压、糖尿病为由拒赔。

→ **分析**：拿出问卷一看，恶性肿瘤疾病保险的问题通常只围绕肿瘤相关问题，没有涉及高血压和糖尿病。客户根本不需要回答 → **不存在未如实告知**，保险公司的拒赔是错误的。

### 情形2：没有问询动作或问询动作错误

保险问询包含两个方面：**问询的问题** + **问询的动作**。

**案例**：王某经销售人员田某介绍投保重疾险。田某通过APP操作投保，遇到投保问题时只笼统地问"身体健康吗？有什么异常吗？住过院吗？"王某认为甲状腺结节不影响生活，医生说没事，就回答"没有"。田某遂全部勾选"否"。

**问题**：
- 销售人员没有**逐项问询**，是操作错误
- 客户未告知的过错不能全部归咎于客户
- 正确的问询方式：**拿着投保单，逐项问询**；客户不明白时做专业解释

**后果**：此类案件很多保险公司会做公平理赔处理，但可能追究销售人员责任。

### 情形3：概括性问询

**法律依据**：《保险法司法解释二》第6条第二款：
> 保险人以投保人违反了对投保单询问表中所列概括性条款的如实告知义务为由请求解除合同的，人民法院不予支持。但该概括性条款有具体内容的除外。

#### 什么是概括性问询？

**典型概括性问询**："除上述之外的其他疾病"——前面列举了具体疾病后，最后加一句"除上述之外的其他疾病"，最后这句属概括性问询，**可以不回答**。

**不是概括性问询**："是否患有以下肿瘤类疾病…除上述之外的其他肿瘤"——有"肿瘤类疾病"的具体限定，**不属于概括性问询，需要告知**。

#### 判断难点

概括性问询**没有统一的客观标准**。不同立场、教育背景、工作经历的人，对同一问询的感受和认知可能不同。

**常见误区**：自己觉得问题范围太广太宽，就认定为概括性问询。

**建议**：审慎判断，不要随意扣"概括性问询"的帽子。
- 含有"除此之外其他"字样且无明确限定的 → 较明确为概括性
- 不含"其他"字样但范围较宽 → 需谨慎，不要轻易定性

## 二、反驳理由二：不影响承保

**法律依据**：《保险法》第16条第二款——未告知必须"**足以影响**保险人决定是否同意承保或者提高保险费率"，保险人才有权解除合同。

### 情形1：未告知事项本身不影响承保

**案例**：王某投保时未告知宫颈内膜息肉住院史。调查发现客户已手术切除，病理为息肉，风险已被彻底消除 → 核保结论为标准体承保 → **不影响承保**，保险公司不能解除合同，应正常赔付。

### 情形2：保险公司已知道却仍承保（弃权与禁止反言）

**法律依据**：《保险法》第16条第四款——保险人在合同订立时已经知道投保人未如实告知的情况的，保险人不得解除合同。

**案例**：王某投保时告知了"腹股沟斜疝住院"，并提交了住院病历。保险公司标准体承保。后因乙型肝炎住院，保险公司以住院病历中有乙肝五项和肝功能异常未单独告知为由拒赔。

→ **分析**：客户把住院病历都交给保险公司了，保险公司作为专业机构，在看到病历后**应当知道**客户的异常健康情况。既然仍标准体承保，说明愿意承担风险，法律不允许反悔。

### 情形3：保险公司自己承诺可以承保

**案例**：客户体检有甲状腺结节二级、甲功正常。在网上投保重疾险时如实回答"有甲状腺结节"→ 进入智能核保 → 回答"结节分级为二级"+"甲功正常"→ 标准体承保。一年后确诊甲状腺癌。

→ **分析**：客户按智能核保规定操作，无任何不实告知。保险公司通过智能核保评估了该风险并同意承保，出险后不能对自己已承诺能承保的风险反悔。（弃权与禁止反言原则）

---

## 三、反驳理由三：两年不可抗辩条款

**法律依据**：《保险法》第16条第三款——自合同成立之日起**超过两年**的，保险人不得解除合同；发生保险事故的，保险人应当承担赔偿或者给付保险金的责任。

### 经典案例

| 项目 | 内容 |
|------|------|
| 投保时间 | 2012年5月11日 |
| 险种 | 终身寿险+重疾险（保额12万） |
| 投保前病史 | 2012年2月曾住院，诊断"颅内占位性病变" |
| 出险 | 2018年4月3日因病去世（保单生效近6年） |

保险公司以未告知投保前病史为由解约拒赔。受益人起诉。

**法院判决**：保险合同成立已超过两年，依据《保险法》第16条第三款，保险公司不得解除合同，应按约定给付保险金。

### 重要提示

两年不可抗辩条款**不是万能条款**：
- 它只针对"未如实告知"这一种情形
- 不适用于先天性疾病等免责条款的范畴
- 具体限制见第六课

## 四、反驳理由四：保险公司的操作失误

### 情形1：30日内未行使合同解除权

**法律依据**：《保险法》第16条第三款——自保险人知道有解除事由之日起，**超过30日不行使而消灭**。

**重要**：这个30天在法律上叫"**除斥期间**"，任何特殊情况（含不可抗力、疫情等）都**不会**让它终止、中断或延长。就是雷打不动的30天。

**起算时间**：保险公司知道或应当知道解除事由之日起开始计算。

### 情形2：只写拒赔，没写解除合同

**法律依据**：《保险法司法解释二》第8条——保险人未行使合同解除权，直接以存在《保险法》第16条第四款、第五款规定的情形为由拒绝赔偿的，人民法院不予支持。

这意味着：拒赔**必须以解除合同为前提**。

### 案例：时间线证据

| 日期 | 事件 |
|------|------|
| 2019年7月22日 | 客户提交理赔申请 |
| 2019年8月2日 | 保险公司获取客户体检报告（得知甲状腺结节未告知） |
| 2019年9月5日 | 保险公司通知拒赔（**仅写拒赔，未写解除合同**） |

→ 从8月3日起算30天，最迟应在9月1日前发出解约通知。9月5日才发通知，且只写拒赔未写解除 → **30天内未行使解除权**，直接帮客户争取回理赔款。

---

## 五、最后一招：争取按"重大过失"处理

如果四大反驳理由都用不上，还有最后的策略：

按照《保险法》第16条第五款，在**重大过失**未告知的情况下，客户仍有获得赔付的可能，最差也能拿回保费。

### 争取界定为"重大过失"的考量因素

1. **投保过程**：未告知的过错责任在谁？
2. **未告知事项距离投保时间**：时间长 → 可能是遗忘
3. **异常情况的严重程度**
4. **客户的文化程度、工作经历**：越不懂越容易获得赔付
5. **未告知事项与出险事故的关联性**

> **特别提醒**：有保险从业经历或医学背景的客户，对其告知义务的要求**高于**普通消费者。

### 关联性上的文章

如果核保结论是除外或加费，可尝试说服保险公司按核保结论处理（保留合同做除外/加费），而非直接解除。

如果确实无法获得赔付，至少帮客户**争取退还保费**，将损失降到最低。

---

## 小结

遇到未如实告知被拒赔时，按以下顺序梳理：

1. **问询瑕疵** → 有没有问？问得对不对？是不是概括性问询？
2. **不影响承保** → 未告知事项是否真正影响核保结论？
3. **两年不可抗辩** → 保单是否生效超过两年？
4. **保险公司操作失误** → 30天内是否行使解除权？通知书是否写了"解除合同"？
5. 如果以上四项都不适用 → **争取按重大过失处理**，最大程度保护客户权益

无论最终结果如何，从专业角度向客户讲清法律依据、进行利弊分析、积极协助与保险公司沟通，客户会理解你的工作，看到你的专业性。
