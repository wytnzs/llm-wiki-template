# QUICKSTART — 5 分钟上手

## 1. 给 Claude Code 发指令

把这段话发给你的 Claude Code：

```
请根据这个 GitHub 仓库，帮我在本地创建一套 AI 知识资产生产系统。
先阅读 README.md、QUICKSTART.md 和 AGENT_SETUP.md。
然后执行初始化流程：
1. 创建目录结构；
2. 复制 templates；
3. 安装 .claude/skills；
4. 创建 CLAUDE.md；
5. 根据我的人群类型选择合适的知识库方案；
6. 生成我的个人知识库使用说明；
7. 不要读取、上传或暴露我的隐私资料。
```

## 2. 开始使用

初始化完成后，对 Claude Code 说：

| 你想做什么 | 说这个 |
|-----------|--------|
| 记录一个想法 | `记一下：XXX` |
| 整理收件箱 | `整理收件箱` |
| 提炼知识点 | `提炼知识卡片` |
| 查知识 | `查一下：XX` |
| 更新索引 | `更新知识地图` |
| 写文章 | `出稿` / `写公众号` |

## 3. 最小闭环

每天只需做这 3 件事：

1. **记一下** — 把遇到的想法、问题、案例随手记到 Inbox
2. **整理收件箱** — 每周清理一次，提炼成卡片
3. **写内容** — 从卡片和主题地图中生成选题
