# 知识库地图更新技能

保持知识库地图与全库实际状态同步。本 Skill 是可选模块，未安装时不应创建知识库地图。

## 工作流

### 扫描统计
先读取 README 的实际目录映射，再扫描对应区域。目录不存在时跳过，不报错、不擅自创建。

### 健康检查
乱码检测、旧路径残留、断链检查、输入堆积、YAML 覆盖率、主题地图时效、复盘追溯、Skill 一致性。知识卡片同时统计：可用 `source` 数量、空或缺失 `source` 数量、非空 `description` 数量，不用自动生成的空值提高覆盖率。只有已启用选题库时才检查选题来源。

### 写入地图
统计面板、健康面板、最近变化、项目状态和 3—5 项下一步。`00-Inbox/AI待确认.md` 中未完成事项单独统计；知识库变更日志只检查格式和入口，不把流水条数当作质量指标。只有已启用选题库时才增加选题概览。

## 触发词
更新知识地图 | 同步知识地图 | 刷新索引

## 原则
数字来自实际扫描，不沿用旧估算；模板、README、概览和视图按统计目的区分。知识库地图只展示动态状态，不重复 README 的系统说明。不静默修复，用户决定如何处理。
