# 知识库地图更新技能

保持知识库地图与全库实际状态同步。本 Skill 是可选模块，未安装时不应创建知识库地图。README 解释系统，知识库地图只展示动态状态。

## 工作流

### 扫描统计
先读取 README 的实际目录映射，再扫描对应区域。目录不存在时跳过，不报错、不擅自创建。

### 健康检查
乱码检测、旧路径残留、断链检查、输入堆积、YAML 最小字段、关系字段形态、来源覆盖、主题页证据和 Skill 一致性。只有已启用选题库时才检查选题来源。

必须区分真实断链、模板占位、待补关系和历史旧路径；不能为了让数字归零而创建空卡片、虚假来源或无意义链接。

### 写入地图
写入当前规模、项目状态、健康问题、最近重要变化和 3 至 5 项下一步。数字必须来自本次实际扫描，不沿用旧地图估算。只有已启用选题库时才增加选题概览。

## 触发词
更新知识地图 | 同步知识地图 | 刷新索引

## 原则
数字来自实际扫描；地图不重复 README；不静默修复；来源不足时保留缺口；用户决定高影响处理。

完整运行顺序见 `05-Skills/定期维护与审计.md`；在模板仓库中对应 `docs/定期维护与审计.md`。
