# 知识库地图更新技能

保持知识库地图与全库实际状态同步。本 Skill 是可选模块，未安装时不应创建知识库地图。

## 工作流

### 扫描统计
先读取 README 的实际目录映射，再扫描对应区域。目录不存在时跳过，不报错、不擅自创建。

### 健康检查
乱码检测、旧路径残留、断链检查、输入堆积、YAML覆盖率、主题地图时效、复盘追溯、Skill一致性。只有已启用选题库时才检查选题来源。

### 写入地图
统计面板、健康面板、更新记录、项目状态。只有已启用选题库时才增加选题概览。

## 触发词
更新知识地图 | 同步知识地图 | 刷新索引

## 原则
数字来自实际扫描，不静默修复，用户决定如何处理。
