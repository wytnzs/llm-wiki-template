# Changelog

## 2026-07-17 - v1.2.0 运作系统与协作边界

- 新增知识库运作系统说明，统一输入、资源、资产、主题、项目和反馈的流转关系
- 明确主题、聚合、标签、双链、主题中心和知识库地图的职责边界
- 新增关系三问法、自然语言索引下钻顺序和批量关系质量标准
- 增加 AI 修改权限、协作边界和 GitHub 公开安全边界
- 新增每周、双周、月度维护与审计运行单
- 初始化结构将旧的“主题聚合”目录统一为“主题中心/主题页”，并增加主题中心入口模板
- 同步更新知识库协作 Skill、主题维护 Skill、知识库地图模块和初始化模板

## 2026-07-13 - v1.1.0 最小标准

- README 成为唯一必需入口，不默认创建 index.md
- 目录由强制结构改为推荐角色映射
- 知识库地图、案例库和选题库改为按需安装
- 选题库安装前必须征求用户意见，不宣称自动生成正式选题
- 文档从八份整合为标准、操作指南和人群适配三份
- 初始化脚本会复制模板和 Skill，并支持可选模块
- 知识文件统一以 `title` 和 `type` 为最小字段，优先补充 `description`

## 2026-07-09 - v1.0.0 初始发布

- 首次发布 LLM Wiki 知识资产生产系统模板
- 包含目录建议、模板、公开版 Skill、示例和初始化脚本
