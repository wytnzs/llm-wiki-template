# Changelog

## 2026-07-13 - v1.1.0 最小标准

- README 成为唯一必需入口，不默认创建 index.md
- 目录由强制结构改为推荐角色映射
- 知识库地图、案例库和选题库改为按需安装
- 选题库安装前必须征求用户意见，不宣称自动生成正式选题
- 文档从八份整合为标准、操作指南和人群适配三份
- 初始化脚本会复制模板和 Skill，并支持可选模块
- 知识文件统一以 `title` 和 `type` 为最小字段，优先补充 `description`

## 2026-07-09 - v1.0.0 初始发布

- 首次发布 LLM Wiki 知识资产生产系统模板
- 包含目录建议、模板、公开版 Skill、示例和初始化脚本
