# 主题中心

主题中心把自然语言问题、主题入口和主题页放在同一个模块中。

- **主题**：需要持续回答的问题；
- **聚合**：围绕问题寻找和筛选资料的动作，不单独形成目录；
- **主题页**：保存当前判断、证据、方法、反例和边界；
- **双链**：从主题页继续下钻到具体卡片、案例或依据；
- **知识库地图**：只记录全库状态，不放在本模块中。

## 使用方式

1. 先用自然语言问题定位最接近的主题；
2. 读取主题页的当前判断和边界；
3. 沿精选链接进入知识卡片或案例；
4. 正式引用时回到原始资料核验。
