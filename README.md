# LLM Wiki — 知识资产生产系统

> 这不是一个笔记模板，而是一套让 AI 帮你把碎片信息加工成知识资产的**生产线**。

## 适合谁

| 人群 | 解决什么问题 |
|------|------------|
| 保险/法律/咨询从业者 | 案例管理 + 专业内容输出 |
| 自媒体/知识博主 | 持续选题 + 多平台内容生产 |
| 创业者/管理者 | 决策记录 + 项目复盘 |
| 学生/研究者 | 文献卡片 + 论文写作 |
| 销售/私域运营 | 客户痛点 + 话术体系 |

## 核心卖点

- **不是资料库，是生产线** — 输入 → 提炼 → 连接 → 输出 → 复盘
- **AI 帮你执行** — Claude Code 按 skill 指令自动处理
- **零数据库** — Obsidian + Markdown + Claude Code，无需搭建
- **标准可迁移** — YAML 元数据 + 双向链接，不被平台锁定

## 目录结构

```
你的知识库/
├── 00-Inbox/         ← 原始输入（想法、剪藏、笔记）
├── 01-Projects/      ← 当前项目
├── 02-Areas/         ← 长期资产（卡片/案例/主题地图）
├── 03-Resources/     ← 外部资料
├── 04-Archive/       ← 归档
├── 06-选题库/        ← 选题资产
└── .claude/skills/   ← AI 执行技能
```

## 快速开始

把下面这段话发给 Claude Code：

```
请根据这个 GitHub 仓库，帮我在本地创建一套 AI 知识资产生产系统。
先阅读 README.md、QUICKSTART.md 和 AGENT_SETUP.md。
然后执行初始化流程：
1. 创建目录结构；
2. 复制 templates；
3. 安装 .claude/skills；
4. 创建 CLAUDE.md；
5. 根据我的人群类型选择合适的知识库方案；
6. 生成我的个人知识库使用说明；
7. 不要读取、上传或暴露我的隐私资料。
```

## 隐私承诺

- 本仓库是公开模板，**不包含任何真实资料**
- 你的私人数据只存在本地，不上传到 GitHub
- `.gitignore` 已配置排除所有私人目录
- 初始化脚本不会读取你的隐私文件
