# 不同人群适配

所有人使用同一条基本链路：输入、整理、连接、使用、复盘。差异只体现在资产类型和可选模块。

| 人群 | 常见资产 | 建议模块 |
|---|---|---|
| 专业服务 | 方法、法规、案例、策略 | 案例库；知识库地图 |
| 内容创作者 | 观点、素材、成品、复盘 | 选题库；知识库地图 |
| 创业者和管理者 | 决策、会议、客户反馈、项目复盘 | 知识库地图 |
| 学生和研究者 | 文献、概念、论点、引用 | 主题地图；知识库地图 |
| 销售和私域运营 | 客户问题、场景、方案、话术 | 案例库；选题库按需 |

## 适配原则

1. 先问用户需要解决什么问题，再决定资产类型；
2. 不因为职业标签自动安装全部模块；
3. 不预建大量空目录；
4. 用户已有结构能工作时，优先保留；
5. 真实客户、案件和业务数据只保存在本地；
6. 选题库只有在用户明确需要内容生产时才安装。
