# 不同人群的知识库方案

不同类型用户对知识资产的需求不同，本系统提供可配置的模块化方案。以下按五类典型用户给出推荐配置。

---

## 1. 专业服务从业者（保险/法律/金融/咨询）

### 核心诉求

高频处理真实案例，需要快速检索事实依据、判断框架、风险边界。

### 转化链

```text
客户案例 → 判例库（脱敏后） → 知识卡片（判断框架） → 主题聚合（当前立场） → 服务交付 / 内容生产
```

### 知识资产重点

| 资产类型 | 重点 |
|---------|------|
| 判例库 | 案件三视图（按类型、按原因、按分类），核心是事实 + 裁判观点 + 攻击/防御角度 |
| 知识卡片 | 判断标准、法律依据、操作流程、风险提示 |
| 主题聚合 | 具体问题的当前立场（例如：关于 A 类风险的应对策略） |
| 选题库 | 常见痛点选题、公众认知误区 |

### 推荐 Skill

- 判例入库自动化
- 知识卡片提炼
- 判例检索与比对
- 主题聚合维护

### 初始化指令

```text
创建判例库目录结构，按三视图维度（类型/原因/分类）建立索引文件。
创建一个示例判例卡片（脱敏通用案例），展示判例录入标准格式。
```

## 2. 内容创作者（自媒体/博主/课程制作人）

### 核心诉求

持续产出内容，需要稳定的选题来源、素材积累、多平台适配。

### 转化链

```text
灵感/素材 → 知识卡片（概念） → 主题聚合（立场） → 选题卡片 → 内容简报 → 多平台生产
```

### 知识资产重点

| 资产类型 | 重点 |
|---------|------|
| 知识卡片 | 核心概念、认知框架、案例（脱敏）、类比 |
| 主题聚合 | 领域内每个议题的当前认知和立场 |
| 选题库 | 选题卡片（痛点+角度+素材+分发建议），选题总索引 |
| 无需 | 判例库（除非内容本身需要） |

### 推荐 Skill

- 选题卡片创建
- 知识卡片提炼
- 主题聚合维护
- 内容简报生成
- 多平台适配（公众号、小红书、短视频）
- 知识地图更新

### 初始化指令

```text
创建选题库目录结构和选题卡片模板。创建知识卡片目录（按内容领域分子目录）。
创建一份完整的知识库地图首页。提供内容简报模板。
```

## 3. 创业者/管理者

### 核心诉求

做决策、管项目、记复盘、沉淀团队经验。

### 转化链

```text
决策思考 / 会议记录 → 决策卡片（判断框架） → 项目管理 → 复盘 → 更新决策卡片
```

### 知识资产重点

| 资产类型 | 重点 |
|---------|------|
| 知识卡片 | 决策框架、商业模型、管理方法论、认知偏误 |
| 项目区 | 每个项目的计划、进展、复盘 |
| 主题聚合 | 每个业务方向的当前判断和策略 |
| 无需 | 判例库、选题库 |

### 推荐 Skill

- 决策记录创建
- 项目复盘总结
- 知识卡片提炼

### 初始化指令

```text
创建项目区目录和项目模板。创建知识卡片目录（商业/管理/认知子目录）。
创建主题聚合目录。提供复盘模板。
```

## 4. 学生/研究者

### 核心诉求

理解概念体系、做文献笔记、产出论文/报告。

### 转化链

```text
阅读/听课 → 文献笔记 → 概念卡片 → 主题聚合 → 输出大纲 → 论文/报告
```

### 知识资产重点

| 资产类型 | 重点 |
|---------|------|
| 知识卡片 | 学科核心概念、理论框架、学者观点 |
| 资源区 | 文献笔记、课程笔记、读书笔记 |
| 主题聚合 | 每个研究主题的当前文献综述和个人理解 |
| 项目区 | 论文/研究报告写作项目 |

### 推荐 Skill

- 知识卡片提炼（从笔记中提取）
- 主题聚合维护
- 文献笔记模板

### 初始化指令

```text
创建知识卡片目录（按学科分子目录）。
创建主题聚合目录。创建资源区目录（读书笔记子目录）。
提供概念卡片模板和文献笔记模板。
```

## 5. 销售/私域运营者

### 核心诉求

管理客户画像、准备话术、跟进线索、保护客户隐私。

### 转化链

```text
客户沟通记录（脱敏） → 客户画像卡片 → 话术卡片 / 异议处理卡片 → 跟进策略 → 下次沟通
```

### 知识资产重点

| 资产类型 | 重点 |
|---------|------|
| 知识卡片 | 客户画像框架、异议处理框架、产品知识、行业分析 |
| 选题库 | 朋友圈内容选题、客户教育选题 |
| 项目区 | 大客户跟进项目、营销活动策划 |
| ⚠️ 隐私保护 | 客户真实信息**绝不**进入知识库 |

### 隐私保护红线

- 客户姓名、联系方式、地址、身份证号等个人身份信息**不得**入库
- 客户案例必须脱敏：使用代号（如"客户A"），删除所有可识别信息
- 销售话术中的客户故事必须改编，不可复制真实对话
- 如果涉及合规要求（金融、医疗等），咨询法务后再建库

### 推荐 Skill

- 知识卡片提炼
- 主题聚合维护
- 选题卡片创建

### 初始化指令

```text
创建知识卡片目录（产品知识/行业分析/销售方法子目录）。
创建选题库目录。提供客户画像卡片模板（不含真实个人信息字段）。
```
